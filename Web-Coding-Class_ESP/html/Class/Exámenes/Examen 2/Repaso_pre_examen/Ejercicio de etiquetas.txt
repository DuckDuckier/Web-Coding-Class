Incluye las siguientes secciones usando etiquetas HTML:




<!DOCTYPE html> para indicar el tipo de documento.
 
al menos pon estas etiquetas.


<html>, <head>, <header>, <body >


En <head>, incluye:




<title> con tu nombre completo.




<meta> para agregar el autor.




<style> para aplicar estilos (puede ser código interno).




<script> con un saludo en JavaScript: alert("Hola, bienvenido a mi página");




En <body>, incluye:




<header> con el título principal (<h1>) de la página.




<nav> con una lista no ordenada <ul> de tus intereses.




<section> con una lista no ordenada <ul> de tus habilidades.




<article> con un párrafo sobre quién eres.




<img> con una foto  (usar un archivo de ejemplo o URL) añadirle el atributo alt descriptivo.








Uso de <br> y <hr> para separar contenido visualmente.




Un elemento <div> que agrupe sobre musica o peliculas favoritas.(estructurar con encabezado y lista ordenada )
<video> sobre algún pasatiempo o presentación (de una película o música).




Una imagen con el atributo title aplicado a un botón o imagen para mostrar texto al pasar el ratón.




<footer> con tu información de contacto o derechos reservados.
