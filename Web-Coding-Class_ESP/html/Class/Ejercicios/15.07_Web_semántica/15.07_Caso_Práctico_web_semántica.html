Explicación de etiquetas utilizadas.

  • <header>: Encabezado principal del sitio web
  • <nav>: Menú de navegación con enlaces
  • <main>: Área principal que contiene todo el contendio visible.
  • <article>: Contenido autónomo, puede incluir varias secciones
  • <section>: Cada bloque temçatico dentro del artículo.
  • <aside>: Información adicional, como explicaciones o enlaces.
  • <footer>: Pie de página con autor, fecha o contacto.

Estas etiquetas permiten que el contenido esté bien organizado y claro tanto para el navegador como para el usuario
