Utilizando únicamente HTML, crea una página web que represente el diseño concreto que se te proporciona (puede ser un boceto en papel o una imagen de referencia). 2. La página debe al menos incluir los siguientes elementos: - Un encabezado principal (h1) con el título de la página. - anclaje del menu hacia las distintas secciones - Una sección de navegación con enlaces ficticios. - Una sección de contenido principal con al menos dos apartados. - Una imagen representativa (puedes usar una imagen libre o un marcador de posición). - Una lista (ordenada o desordenada) relacionada con el contenido. - Un pie de página con información de contacto ficticia (Email, teléfono, etc.). 3.
