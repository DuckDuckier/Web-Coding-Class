Expresiones Regulares: Símbolos y Ejemplos

Símbolo Significado Ejemplo de uso Qué valida
. Cualquier carácter (excepto salto de línea) a. "ab", "a1", "a@"
\d Dígito (0–9) \d{3} Tres dígitos: "123"
\D No dígito \D+ Letras o símbolos: "abc", "@"
\w Carácter alfanumérico o guion bajo \w{5,} Palabras como "user_1", "abc12"
\W No alfanumérico \W+ "@#", "!"
\s Espacio en blanco \s " " (espacio), tabulación
\S No espacio en blanco \S+ "texto", "123"
^ Inicio de la cadena ^abc Comienza con "abc"
$ Fin de la cadena abc$ Termina con "abc"
* 0 o más repeticiones a* "", "a", "aaa"
+ 1 o más repeticiones a+ "a", "aa", pero no ""
? 0 o 1 repetición a? "", "a"
{n} Exactamente n repeticiones \d{4} "2025"
{n,m} Entre n y m repeticiones [A-Z]{2,4} "AB", "ABC", "ABCD"
[] Conjunto de caracteres [aeiou] Una vocal
[^] Exclusión de caracteres [^0-9] No dígitos
() Agrupación (abc)+ "abc", "abcabc"
| Alternancia (OR) cat|dog "cat" o "dog"
(?=...) Búsqueda anticipada positiva (?=.*\d) Al menos un número
(?!...) Búsqueda anticipada negativa (?!abc) No contiene "abc"

Ejemplos prácticos

1. Validar un número de 4 cifras:
^\d{4}$
// Solo acepta cadenas de exactamente 4 dígitos, como "2025"
2. Validar una palabra que no contenga números:
^[^\d]+$
// Solo letras o símbolos, no acepta números
3. Validar si contiene al menos una letra y un número:
^(?=.*[A-Za-z])(?=.*\d).+$
// Debe tener al menos una letra y un número
4. Validar que NO contenga la palabra "abc":
^(?!.*abc).*$
// No acepta cadenas que contengan "abc"
