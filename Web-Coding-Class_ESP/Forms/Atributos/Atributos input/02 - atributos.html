Tabla de tipos de <input> en HTML

Tipo de Input ¿Qué hace? Ejemplo HTML básico
text Campo para escribir texto libre
password Campo para contraseñas (oculta los caracteres)
checkbox Casilla para marcar opciones múltiples
radio Seleccionar una sola opción entre varias Opción 1
Opción 2
submit Botón que envía el formulario
reset Botón que reinicia todos los campos
button Botón simple (requiere JavaScript)
file Seleccionar y subir archivos
hidden Campo invisible para datos internos (no visible)
image Usa una imagen como botón
email Campo para correos electrónicos (valida formato)
number Campo para ingresar solo números
date Selector de fecha
color Selector de color
range Barra deslizante para valores
tel Campo para números de teléfono
url Campo para direcciones web (valida formato)
