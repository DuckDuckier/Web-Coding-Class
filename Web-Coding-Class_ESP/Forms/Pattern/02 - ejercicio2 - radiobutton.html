Radio Buttons en HTML

Los radio buttons son elementos de formulario que permiten al usuario seleccionar una sola opción entre varias. Todos los botones que comparten el mismo atributo name forman parte del mismo grupo. Cuando el usuario selecciona uno y envía el formulario, se envía el value del botón elegido. Son útiles para preguntas de opción única como género, método de pago o nivel educativo.
