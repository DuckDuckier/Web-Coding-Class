¿Cómo se divide un CMS?

Frontend (parte visible)

  • Diseño y estructura de la web
  • Contenido que ve el visitante
  • Interacción con menús, botones y formularios
  • Lenguajes: HTML, CSS, JavaScript

Backend (parte interna)

  • Panel de administración del sitio
  • Gestión de contenido, usuarios y configuración
  • Acceso restringido a usuarios con permisos
  • Lenguajes: PHP, SQL, Python
