Comparación: Página Estática vs Página Dinámica

¿Qué es una página estática?

Una página estática muestra siempre el mismo contenido. Está escrita en HTML y CSS, sin interacción con bases de datos ni scripts del lado del servidor.

<h1>Bienvenido a mi sitio web</h1> <p>Esta es una página estática. Siempre verás este mismo contenido.</p>

¿Qué es una página dinámica?

Una página dinámica genera contenido en tiempo real, dependiendo de la interacción del usuario o de datos externos. Utiliza tecnologías como JavaScript, PHP o bases de datos.

<p>Hola, Joaquín. Gracias por tu mensaje. Te responderemos pronto.</p>

Definición

Interactividad de una web

La página web permite que el usuario interactúe con la información que muestra. Esta interacción se realiza, por ejemplo, a través de formularios que recogen datos como nombre, correo o comentarios.
