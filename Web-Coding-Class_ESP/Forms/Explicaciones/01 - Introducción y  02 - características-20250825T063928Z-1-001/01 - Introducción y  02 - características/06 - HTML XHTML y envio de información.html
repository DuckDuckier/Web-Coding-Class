HTML, XHTML y Envío de Información

¿Qué es una página web HTML?

Una página HTML es un documento que estructura el contenido de una web usando etiquetas como títulos, párrafos, imágenes y enlaces.

<!DOCTYPE html> <html> <head><title>Mi página</title></head> <body> <h1>Bienvenido</h1> <p>Esta es una página web en HTML.</p> </body> </html>

¿Qué es una página web XHTML?

XHTML es una versión más estricta de HTML que sigue las reglas de XML. Requiere que el código esté perfectamente estructurado.

  • Las etiquetas deben cerrarse correctamente
  • Los atributos van entre comillas
  • Todo debe estar en minúsculas
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd"> <html xmlns="http://www.w3.org/1999/xhtml"> <head><title>Mi página XHTML</title></head> <body> <h1>Bienvenido</h1> <p>Esta es una página web en XHTML.</p> </body> </html>

¿Cómo se envía información al servidor?

Cuando el usuario rellena un formulario y hace clic en "Enviar", los datos se envían al servidor mediante una petición HTTP. El servidor los procesa y responde.

  • El navegador envía los datos con el método POST o método GET
  • El servidor puede guardar, validar o responder con otra página
