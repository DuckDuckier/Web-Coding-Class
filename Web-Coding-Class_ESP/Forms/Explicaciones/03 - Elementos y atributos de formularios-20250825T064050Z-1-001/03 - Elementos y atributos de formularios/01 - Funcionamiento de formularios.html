¿Cómo funcionan los formularios en HTML?

📩 ¿Qué es un formulario?

Un formulario permite al usuario enviar información al servidor. Se usa para registrarse, iniciar sesión, enviar mensajes, hacer pedidos, etc.

🔧 Elementos principales del formulario

  • <form>: Contenedor del formulario
  • <input>: Campo de entrada (texto, email, contraseña, etc.)
  • <label>: Etiqueta descriptiva para cada campo
  • <select>: Menú desplegable
  • <textarea>: Área de texto grande
  • <button> o <input type="submit">: Botón para enviar

⚙️ Atributos comunes

  • name: Identifica el campo para el servidor
  • type: Define el tipo de entrada (text, email, password...)
  • placeholder: Texto de ayuda dentro del campo
  • required: Obliga a rellenar el campo
  • value: Valor por defecto
