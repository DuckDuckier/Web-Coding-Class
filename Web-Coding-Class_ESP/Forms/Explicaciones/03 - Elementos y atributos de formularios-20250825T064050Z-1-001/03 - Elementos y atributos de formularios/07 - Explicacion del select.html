Menú desplegable con <select>

¿Cómo funciona?

El usuario elige una opción visible. El servidor recibe el valor definido en el atributo value de la opción seleccionada.

Ejemplo práctico

Ejemplo de código

<select name="motivo">
  <option value="">-- Selecciona --</option>
  <option value="consulta">Consulta</option>
  <option value="soporte">Soporte técnico</option>
  <option value="otros">Otros</option>
</select>
