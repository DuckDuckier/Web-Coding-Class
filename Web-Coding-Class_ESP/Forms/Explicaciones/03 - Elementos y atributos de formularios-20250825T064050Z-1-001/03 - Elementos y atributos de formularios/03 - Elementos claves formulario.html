Elemento <form> en HTML

¿Qué es <form>?

Es el contenedor principal de cualquier formulario. Dentro de él se colocan todos los campos que el usuario debe rellenar.

Atributos principales

  • action: define la URL a la que se enviarán los datos a un servidor o en un archivo local
  • method: define el método de envío get o post
<form action="procesar.php" method="post"> </form>

Sin acción - vació o "#"

Si no se especifica action, el formulario se envía a la misma página:

<form action="#" method="post">

Acción sin redirección externa

El formulario se procesa localmente, ya sea con JavaScript o con código embebido en la misma página.

¿Por qué se usa?

  • Para evitar recargar o cambiar de página.
  • Para mostrar resultados o mensajes en el mismo lugar.
  • Para validar datos sin enviar a un servidor externo.
