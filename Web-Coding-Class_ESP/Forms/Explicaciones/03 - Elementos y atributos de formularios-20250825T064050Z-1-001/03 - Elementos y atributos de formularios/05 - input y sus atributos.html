Elemento <input> y sus atributos

¿Qué es <input>?

Es el campo de entrada donde el usuario puede escribir información. Se usa para texto, correos, contraseñas, números, fechas, etc.

Atributos principales

  • type: define el tipo de dato (text, email, password, number...)
  • name: identifica el campo para el servidor
  • placeholder: muestra un texto de ayuda dentro del campo
  • required: obliga a rellenar el campo antes de enviar

Ejemplo práctico

Ejemplo de código

<input type="text" name="nombre" placeholder="Escribe tu nombre" required>
<input type="email" name="correo" placeholder="ejemplo@correo.com" required>
<input type="number" name="edad" placeholder="18" required>
