21. El atributo action de un formulario indica el tipo de datos que se enviarán.

22. El elemento <input> puede tener diferentes tipos como texto, contraseña o botón.

23. El atributo method="get" oculta los datos enviados en el cuerpo de la solicitud.

24. El atributo required obliga al usuario a completar un campo antes de enviar el formulario.

25. El elemento <textarea> se usa para campos de texto de una sola línea.

26. Puedes usar <label> para mejorar la accesibilidad de los formularios.

27. El atributo name no es necesario si usas JavaScript para procesar el formulario.

28. <input type="email"> valida automáticamente que el texto tenga formato de correo.

29. El botón de envío se crea con <input type="submit"> o <button type="submit">.

30. El atributo readonly impide que el usuario modifique el campo y también evita que se copie su contenido.
