Test de Formularios HTML - Parte 1

1. El atributo autocomplete permite que el navegador sugiera valores previamente ingresados.

2. El atributo disabled permite enviar el valor del campo aunque esté deshabilitado.

3. Un formulario puede tener múltiples botones de envío (submit).

4. El evento onblur se activa cuando el usuario presiona un botón.

5. El atributo name es esencial para que los datos del campo se envíen correctamente.

6. El método reset() en JavaScript borra todos los datos del formulario.

7. El atributo readonly impide que el usuario modifique el campo, pero permite copiar su contenido.

8. El atributo action define el tipo de datos que se enviarán en el formulario.

9. El evento oninput se activa cada vez que el valor de un campo cambia.

10. El atributo enctype="multipart/form-data" se usa para enviar archivos en formularios.
