Lección 1: Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Controla aspectos como colores, fuentes, espaciados y disposición de los elementos.

¿Cómo funciona?

CSS funciona aplicando reglas de estilo a elementos HTML. Cada regla tiene dos partes principales:

  • Selector: Indica a qué elementos HTML se aplica el estilo
  • Declaración: Define qué propiedades se modifican y sus valores

Ejemplo de sintaxis CSS:

selector {
    propiedad: valor;
    propiedad: valor;
}

Ejercicio 1: Crear tu primera regla CSS

Instrucciones: Crea una regla CSS que:

  1. Seleccione todos los párrafos (<p>)
  2. Cambie el color del texto a azul
  3. Establezca el tamaño de fuente a 18px

Solución:

p {
    color: blue;
    font-size: 18px;
}

Explicación: El selector "p" aplica el estilo a todos los párrafos. La propiedad "color" cambia el color del texto, y "font-size" establece el tamaño de la fuente.

Lección 2: Selectores Básicos

Tipos de selectores:

  • Selector de elemento: Selecciona todos los elementos de un tipo (p, div, h1, etc.)
  • Selector de clase: Selecciona elementos con un atributo class específico (usa .)
  • Selector de ID: Selecciona un elemento con un atributo id único (usa #)
  • Selector universal: Selecciona todos los elementos (usa *)

Ejemplos de selectores:

/* Selector de elemento */
h1 { color: green; }

/* Selector de clase */
.destacado { background: yellow; }

/* Selector de ID */
#principal { border: 1px solid black; }

/* Selector universal */
* { margin: 0; }

Lección 3: Colores y Fondos

Trabajando con colores:

CSS ofrece varias formas de definir colores:

  • Nombres de colores: red, blue, green, etc.
  • Hexadecimal: #FF0000 (rojo), #00FF00 (verde)
  • RGB: rgb(255, 0, 0) (rojo)
  • RGBA: rgb(255, 0, 0, 0.5) (rojo con 50% transparencia)

Propiedades de fondo:

  • background-color: Color de fondo
  • background-image: Imagen de fondo
  • background-repeat: Cómo se repite la imagen
  • background-position: Posición de la imagen

Ejemplos de colores y fondos:

/* Diferentes formas de definir color */
.rojo { color: red; }
.verde { color: #00FF00; }
.azul { color: rgb(0, 0, 255); }
.transparente { color: rgba(0, 0, 255, 0.7); }

/* Propiedades de fondo */
.fondo-simple {
    background-color: lightblue;
}

.fondo-imagen {
    background-image: url('imagen.jpg');
    background-repeat: no-repeat;
    background-position: center;
}

Lección 4: Modelo de Caja (Box Model)

¿Qué es el Box Model?

En CSS, cada elemento se considera como una caja con cuatro componentes:

  1. Contenido: El contenido real del elemento (texto, imagen, etc.)
  2. Padding: Espacio alrededor del contenido (dentro del borde)
  3. Borde: Línea que rodea el padding y el contenido
  4. Margin: Espacio fuera del borde (separación con otros elementos)

Propiedades clave:

  • width/height: Ancho y alto del contenido
  • padding: Relleno interno
  • border: Borde
  • margin: Margen externo
  • box-sizing: Define cómo se calcula el tamaño total

Ejemplo de Box Model:

.caja {
    width: 300px;
    height: auto;
    padding: 20px;
    border: 5px solid #3498db;
    margin: 15px;
    box-sizing: border-box; /* Incluye padding y border en el tamaño total */
}

Ejercicio 4: Creando un botón con Box Model

Instrucciones: Diseña un botón con:

  1. Ancho de 200px y alto de 50px
  2. Padding de 15px horizontal y 10px vertical
  3. Borde de 2px sólido color verde
  4. Margin de 10px
  5. Usa box-sizing: border-box

Solución:

.boton {
    width: 200px;
    height: 50px;
    padding: 10px 15px;
    border: 2px solid green;
    margin: 10px;
    box-sizing: border-box;
    background-color: #f0f8f0;
    text-align: center;
    line-height: 30px; /* Centrar verticalmente el texto */
}

Explicación: Con box-sizing: border-box, el padding y el borde se incluyen dentro del ancho y alto especificados. El line-height ayuda a centrar el texto verticalmente.

Lección 5: Texto y Fuentes

Propiedades de texto:

  • color: Color del texto
  • font-size: Tamaño de la fuente
  • font-family: Tipo de letra
  • font-weight: Grosor de la fuente (normal, bold)
  • text-align: Alineación del texto (left, center, right, justify)
  • text-decoration: Decoración (none, underline, etc.)
  • line-height: Altura de línea
  • letter-spacing: Espaciado entre letras

Fuentes web:

Puedes usar fuentes personalizadas con @font-face o importar desde Google Fonts.

Ejemplos de propiedades de texto:

.titulo {
    font-family: 'Arial', sans-serif;
    font-size: 32px;
    font-weight: bold;
    text-align: center;
    color: #2c3e50;
    text-transform: uppercase;
}

.parrafo {
    font-family: 'Georgia', serif;
    font-size: 16px;
    line-height: 1.6;
    color: #555;
    text-align: justify;
    letter-spacing: 0.5px;
}

.enlace {
    color: #3498db;
    text-decoration: none;
    font-weight: bold;
}

.enlace:hover {
    text-decoration: underline;
}

Lección 6: Posicionamiento

Tipos de posicionamiento:

  • Static: Posicionamiento por defecto (flujo normal)
  • Relative: Se posiciona relativamente a su posición normal
  • Absolute: Se posiciona respecto al ancestro posicionado más cercano
  • Fixed: Se posiciona respecto a la ventana del navegador
  • Sticky: Comporta como relative hasta alcanzar un punto, luego se vuelve fixed

Propiedades clave:

  • position: Define el tipo de posicionamiento
  • top, right, bottom, left: Desplazamiento desde los bordes
  • z-index: Orden de apilamiento (eje Z)

Ejemplos de posicionamiento:

/* Posicionamiento relativo */
.relativo {
    position: relative;
    top: 20px;
    left: 30px;
}

/* Posicionamiento absoluto */
.contenedor {
    position: relative;
    width: 300px;
    height: 200px;
}

.absoluto {
    position: absolute;
    top: 10px;
    right: 15px;
}

/* Posicionamiento fijo */
.fijo {
    position: fixed;
    bottom: 0;
    right: 0;
    background: #333;
    color: white;
    padding: 10px;
}

Ejercicio 6: Creando un menú flotante

Instrucciones: Diseña un menú que:

  1. Esté fijo en la parte superior de la página
  2. Tenga fondo oscuro y texto claro
  3. Ocupe todo el ancho de la página
  4. Tenga z-index: 1000 para que esté sobre otros elementos

.menu-flotante {

    top: 0;
    left: 0;

    margin: 0;

.menu-flotante a {
    color: white;
    text-decoration: none;
    font-weight: bold;

Lección 7: Flexbox

¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un modelo de diseño unidimensional que permite organizar elementos en un contenedor de manera eficiente, incluso cuando su tamaño es desconocido o dinámico.

Conceptos clave:

  • Contenedor flexible: Elemento con display: flex
  • Ítems flexibles: Elementos hijos directos del contenedor
  • Eje principal: Dirección principal de los ítems (definida por flex-direction)
  • Eje secundario: Perpendicular al eje principal

Propiedades principales:

  • display: flex; Activa el modo flexbox
  • flex-direction: Dirección de los ítems (row, column)
  • justify-content: Alineación en el eje principal
  • align-items: Alineación en el eje secundario
  • flex-wrap: Permitir que los ítems se multi-línea
  • flex-grow: Habilidad de un ítem para crecer

Ejemplos de Flexbox:

/* Contenedor flex básico */
.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

/* Dirección columna */
.flex-column {
    display: flex;
    flex-direction: column;
    align-items: stretch;
}

/* Ítems flexibles */
.flex-item {
    flex: 1; /* Todos los ítems crecen igual */
    margin: 5px;
}

/* Ítem que ocupa más espacio */
.flex-item-grow {
    flex: 2; /* Ocupa el doble que los otros */
}

Ejercicio 7: Creando una galería de imágenes

Instrucciones: Diseña una galería con:

  1. Contenedor con display: flex
  2. Las imágenes se distribuyan en fila con espacio entre ellas
  3. Las imágenes tengan tamaño flexible (flex: 1)
  4. En pantallas pequeñas, las imágenes se apilen en columna

Solución:

.galeria {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
    gap: 15px;
}

.galeria img {
    flex: 1 1 300px; /* Crecer, encoger, base 300px */
    max-width: 100%;
    height: auto;
    border-radius: 8px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

/* Para pantallas pequeñas */
@media (max-width: 768px) {
    .galeria {
        flex-direction: column;
    }
    
    .galeria img {
        width: 100%;
    }
}

Explicación: Flexbox permite crear diseños responsivos fácilmente. Con flex-wrap: wrap, los ítems pueden pasar a la siguiente línea si no hay espacio. La media query cambia la dirección a columna en pantallas pequeñas.

Lección 8: Responsive Design

¿Qué es el diseño responsivo?

Es el enfoque de diseño web que hace que las páginas se vean bien en todos los dispositivos y tamaños de pantalla, desde móviles hasta escritorios.

Técnicas principales:

  • Media Queries: Aplicar estilos según características del dispositivo
  • Unidades relativas: Usar %, em, rem, vw, vh en lugar de píxeles fijos
  • Imágenes flexibles: max-width: 100% para imágenes
  • Mobile-first: Diseñar primero para móviles y luego para pantallas más grandes

Media Queries:

Permiten aplicar estilos específicos según:

  • Ancho de pantalla (width, min-width, max-width)
  • Altura de pantalla (height, min-height, max-height)
  • Orientación (orientation: portrait/landscape)
  • Resolución (resolution)

Ejemplos de Responsive Design:

/* Estilos base para móviles */
.contenedor {
    width: 100%;
    padding: 15px;
    font-size: 16px;
}

/* Para tablets */
@media (min-width: 768px) {
    .contenedor {
        max-width: 750px;
        margin: 0 auto;
        font-size: 18px;
    }
}

/* Para escritorios */
@media (min-width: 992px) {
    .contenedor {
        max-width: 970px;
        font-size: 20px;
    }
}

/* Imágenes responsivas */
img {
    max-width: 100%;
    height: auto;
}

/* Menú responsivo */
.menu {
    display: flex;
    flex-direction: column;
}

@media (min-width: 768px) {
    .menu {
        flex-direction: row;
        justify-content: space-between;
    }
}

Ejercicio 8: Diseñando una página responsiva

Instrucciones: Crea una estructura responsiva con:

  1. Header con logo y menú (en móviles: menú hamburguesa, en escritorio: menú horizontal)
  2. Contenido principal con sidebar (en móviles: sidebar debajo, en escritorio: a la derecha)
  3. Footer que siempre esté en la parte inferior
  4. Usa media queries para los breakpoints: 768px y 1024px

Solución:

/* Estructura base */
body {
    margin: 0;
    font-family: Arial, sans-serif;
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

header {
    background: #2c3e50;
    color: white;
    padding: 15px;
}

.logo {
    font-size: 24px;
    font-weight: bold;
}

.menu-toggle {
    display: block;
    background: none;
    border: none;
    color: white;
    font-size: 24px;
    cursor: pointer;
}

nav {
    display: none;
}

nav ul {
    list-style: none;
    padding: 0;
    margin: 10px 0 0 0;
}

nav li {
    margin: 10px 0;
}

main {
    flex: 1;
    display: flex;
    flex-direction: column;
    padding: 20px;
}

.contenido {
    flex: 1;
}

.sidebar {
    background: #f8f9fa;
    padding: 15px;
    margin-top: 20px;
}

footer {
    background: #34495e;
    color: white;
    text-align: center;
    padding: 15px;
}

/* Tablets */
@media (min-width: 768px) {
    .menu-toggle {
        display: none;
    }
    
    nav {
        display: block;
    }
    
    nav ul {
        display: flex;
        margin: 0;
    }
    
    nav li {
        margin: 0 15px;
    }
    
    main {
        flex-direction: row;
    }
    
    .sidebar {
        margin-top: 0;
        margin-left: 20px;
        width: 250px;
    }
}

/* Escritorios */
@media (min-width: 1024px) {
    header {
        display: flex;
        justify-content: space-between;
        align-items: center;
    }
    
    .contenedor {
        max-width: 1200px;
        margin: 0 auto;
    }
}

Explicación: Este diseño sigue el enfoque mobile-first. En móviles, el menú está oculto y se muestra con un botón. En tablets, el menú se vuelve horizontal y el sidebar se mueve a la derecha. En escritorios, se optimiza el espacio con un contenedor máximo y alineación del header.
