Ejercicio: Organización de Ficheros de Estilo CSS

Objetivo: Aprender a crear, organizar y vincular hojas de estilo CSS externas.

  1. Crea la estructura de archivos:
    • Crea una carpeta llamada mi-web.
    • Dentro, crea estos archivos:
      • index.html
      • reset.css
      • layout.css
      • typography.css
      • buttons.css
      • responsive.css
  2. Escribe el HTML base en index.html: 
    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Web con Estilos</title>
    <!-- Vinculación de estilos -->
    <link rel="stylesheet" href="reset.css">
    <link rel="stylesheet" href="layout.css">
    <link rel="stylesheet" href="typography.css">
    <link rel="stylesheet" href="buttons.css">
    <link rel="stylesheet" href="responsive.css">
</head>
<body>
    <header>
        <h1>Bienvenido a Mi Web</h1>
        <button>Explorar</button>
    </header>
    <main>
        <p>Esta es una página con estilos organizados.</p>
    </main>
    <footer>
        <p>© 2025 Mi Sitio</p>
    </footer>
</body>
</html>
  3. Crea los ficheros de estilo con el siguiente contenido:
    • reset.css: 
      * {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
    • layout.css: 
      body {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
    background: #f8f8f8; /* Cambia este color para practicar */
}
main {
    flex: 1;
    padding: 20px;
}
    • typography.css: 
      body {
    font-family: 'Segoe UI', sans-serif;
    font-size: 16px;
    color: #333;
}
h1 {
    font-size: 2em;
    color: navy;
}
    • buttons.css: 
      button {
    background-color: navy;
    color: white;
    padding: 10px 20px;
    border: none;
    cursor: pointer;
}
button:hover {
    background-color: darkblue;
}
    • responsive.css: 
      @media (max-width: 600px) {
    h1 {
        font-size: 1.5em;
        text-align: center; /* Añade esto para centrar el texto en móviles */
    }
    button {
        width: 100%;
    }
    main, header, footer {
        text-align: center;
    }
}
Actividades para practicar:
  1. Comprueba que los estilos se aplican correctamente en index.html.
  2. Cambia el color de fondo en layout.css y observa el resultado.
  3. Modifica responsive.css para que el texto se centre en móviles (ya incluido arriba).
  4. Crea una nueva página contacto.html con contenido propio y vincula los mismos estilos.
  5. Crea un fichero theme-dark.css con estos estilos para modo oscuro: 
    body {
    background: #181c20;
    color: #eee;
}
h1 {
    color: #7ecfff;
}
button {
    background: #222;
    color: #7ecfff;
}
button:hover {
    background: #333;
}
    Añade este enlace en tu index.html para probar el modo oscuro: 
    <link rel="stylesheet" href="theme-dark.css" media="(prefers-color-scheme: dark)">

¡Experimenta cambiando los estilos y observa cómo afectan a tu web!
