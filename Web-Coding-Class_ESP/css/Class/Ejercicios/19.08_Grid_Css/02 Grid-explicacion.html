Propiedades de CSS Grid

Explicación visual y práctica para aprender diseño con cuadrículas

1. display: grid

Activa el sistema de cuadrícula en el contenedor. Es el punto de partida para usar Grid.

.container {
display: grid;
}

2. grid-template-columns / grid-template-rows

Define cuántas columnas y filas tendrá la cuadrícula. Se pueden usar unidades como px, % o fr.

.container {
grid-template-columns: 1fr 2fr 1fr;
grid-template-rows: 100px 200px;
}

3. gap

Define el espacio entre filas y columnas. También se puede usar row-gap y column-gap.

.container {
gap: 20px;
}

4. grid-column / grid-row

Permite que un elemento ocupe varias columnas o filas.

.item {
grid-column: 1 / 3;
grid-row: 1 / 2;
}

5. grid-template-areas

Permite nombrar zonas de la cuadrícula para facilitar el posicionamiento de los elementos.

.container {
grid-template-areas:
"header header"
"sidebar content"
"footer footer";
}

6. justify-items / align-items

Controlan la alineación de los elementos dentro de sus celdas (horizontal y vertical).

.container {
justify-items: center;
align-items: center;
}

7. justify-content / align-content

Alinean todo el conjunto de la cuadrícula dentro del contenedor.

.container {
justify-content: space-between;
align-content: center;
}
