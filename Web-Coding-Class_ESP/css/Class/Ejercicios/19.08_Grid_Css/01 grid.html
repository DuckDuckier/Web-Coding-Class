Presentación: Introducción a CSS Grid

¿Qué es CSS Grid?

CSS Grid es una técnica moderna de diseño que permite organizar elementos en filas y columnas. Es ideal para crear diseños responsivos y estructurados.

Ejercicio 1: Cuadrícula básica

Usamos display: grid y definimos columnas con grid-template-columns.

.grid-example {
display: grid;
grid-template-columns: 1fr 1fr;
gap: 10px;
}

Podemos usar unidades como px y fr para definir el tamaño de cada columna.

Ejercicio 3: Posicionar elementos

Usamos grid-column y grid-row para que un elemento ocupe más espacio.

.item1 {
grid-column: 1 / 3;
grid-row: 1 / 2;

Ejercicio 4: Grid responsive

Usamos auto-fit y minmax para que las columnas se adapten al ancho disponible.
