¿Qué es un contenedor Grid?

Un contenedor Grid es cualquier etiqueta HTML que agrupa elementos y tiene la propiedad display: grid en CSS.

Esto permite organizar los elementos hijos en filas y columnas de forma ordenada y flexible.

En este ejemplo usamos un <div> como contenedor, pero también podríamos usar <section>, <main> o cualquier otra etiqueta que agrupe contenido.
