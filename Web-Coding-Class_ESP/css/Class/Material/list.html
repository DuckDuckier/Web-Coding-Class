Ejercicio: Propiedades CSS para Listas

Enunciado:
Crea tres listas diferentes utilizando HTML y aplica las siguientes propiedades CSS para listas:

  • list-style-type: Cambia el tipo de viñeta o numeración.
  • list-style-position: Modifica la posición de la viñeta respecto al contenido.
  • list-style-image: Usa una imagen personalizada como viñeta.
  • list-style: Utiliza la propiedad abreviada para combinar varias propiedades.
  • Personaliza el espaciado (padding, margin), color y fondo de las listas.

Objetivo: Comprender y aplicar todas las propiedades CSS relacionadas con listas.



Ejemplo de listas:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  • Elemento uno
  • Elemento dos
  • Elemento tres
  • Elemento A
  • Elemento B
  • Elemento C

Tu tarea: Crea tus propias listas y experimenta con todas las propiedades mencionadas. Cambia los valores y observa cómo afectan la apariencia de las listas.
