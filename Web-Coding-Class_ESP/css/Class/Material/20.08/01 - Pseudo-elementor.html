CSS: Selectores, Pseudo-clases y Pseudo-elementos

Selector CSS

Un selector es la parte del código CSS que indica a qué elementos HTML se aplicarán los estilos.

  • p → selecciona todos los párrafos.
  • .importante → selecciona todos los elementos con la clase importante.
  • #especial → selecciona el elemento con el ID especial.
  • ul > li → selecciona los li hijos directos de un ul.

Función: conectar el CSS con el HTML para aplicar estilos específicos.

Pseudo-clase

Una pseudo-clase representa un estado especial o dinámico de un elemento.

  • :hover → cuando el cursor pasa por encima.
  • :focus → cuando el elemento recibe foco.
  • :nth-child(2) → selecciona el segundo hijo.
  • :checked → cuando un checkbox está marcado.

Función: aplicar estilos según el comportamiento o posición del elemento.

Pseudo-elemento

Un pseudo-elemento permite estilizar partes específicas de un elemento.

  • ::first-letter → estiliza la primera letra.
  • ::first-line → estiliza la primera línea.
  • ::before → inserta contenido antes.
  • ::after → inserta contenido después.

Función: acceder y estilizar fragmentos internos o virtuales del contenido.
