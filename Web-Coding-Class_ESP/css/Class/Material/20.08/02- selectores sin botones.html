Ejemplos de Selectores CSS

  1. Selector de etiqueta: p { color: blue; }
    Selecciona todos los párrafos y los pone en azul.

    Este texto es azul porque está en un <p>.

  2. Selector de clase: .importante { font-weight: bold; }
    Selecciona los elementos con la clase "importante" y los pone en negrita.
    Este texto es importante.
  3. Selector de ID: #especial { background: yellow; }
    Selecciona el elemento con id "especial" y le pone fondo amarillo.
    Este div es especial.
  4. Selector descendiente: article span { color: red; }
    Selecciona los <span> dentro de <article> y los pone en rojo.
    Este span está dentro de un article.
  5. Selector hijo directo: ul > li { background: #eee; }
    Selecciona los <li> hijos directos de <ul> y les pone fondo gris claro.
    • Elemento hijo directo
      1. Este no es hijo directo
  6. Selector de atributo: input[type="text"] { border: 2px solid green; }
    Selecciona los inputs de tipo texto y les pone borde verde.
  7. Selector de grupo: h1, h2 { text-transform: uppercase; }
    Selecciona h1 y h2 y los pone en mayúsculas.

    Este es un h2

  8. Selector de hermano adyacente: h3 + p { color: orange; }
    Selecciona el <p> que sigue inmediatamente a un <h3> y lo pone naranja.

    Título

    Este párrafo es naranja.

    Este no lo es.

  9. Selector de hermano general: h4 ~ p { font-style: italic; }
    Selecciona todos los <p> que son hermanos de un <h4> y los pone en cursiva.

    Subtítulo

    Este párrafo es cursiva.

    Este también.

  10. Selector universal: * { box-sizing: border-box; }
    Selecciona todos los elementos y les aplica box-sizing.
    Todos los elementos tienen box-sizing: border-box.
  11. Pseudo-clase :hover: button:hover { background: lightblue; }
    Cambia el fondo del botón al pasar el ratón.
  12. Pseudo-elemento ::first-letter: .primera-letra::first-letter { font-size: 2em; color: purple; }
    Pone la primera letra más grande y morada.

    La primera letra es especial.
