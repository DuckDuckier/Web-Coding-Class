Ejercicio: Errores HTTP en Archivos Web

Elige 3 sitios web y analiza los errores HTTP que podrían surgir al acceder a distintos archivos. Completa la tabla con el tipo de error, el archivo que lo genera, el nombre del error y una posible solución.

Web Tipo de Error Archivo Nombre del Error Posible Solución
www.ejemplo1.com 404 Not Found /cursos/python.html Página no encontrada Verificar que el archivo existe y que la URL esté bien escrita
www.ejemplo2.com 500 Internal Server Error /api/inscripcion.php Error interno del servidor Revisar el código del backend y los logs del servidor
www.ejemplo3.com 403 Forbidden /admin/panel.html Acceso prohibido Comprobar permisos del archivo y configuración del servidor
