Tipos de Datos en SQL

Tipo Descripción Ejemplo Uso común
INT Número entero 25 Edad, cantidad, ID
DECIMAL(x,y) Número con decimales 19.99 Precios, medidas
VARCHAR(n) Texto limitado 'Juan' Nombres, correos
TEXT Texto largo 'Este producto es...' Descripciones, comentarios
DATE Fecha (AAAA-MM-DD) 2025-09-26 Fechas de nacimiento, registro
TIME Hora (HH:MM:SS) 14:30:00 Horarios, duración
TIMESTAMP Fecha y hora automática 2025-09-26 14:30:00 Registro de eventos
BOOLEAN Verdadero o falso (1 o 0) 1 Activo/Inactivo, Sí/No
ENUM Lista de valores definidos 'masculino', 'femenino' Género, estado
AUTO_INCREMENT Incremento automático 1, 2, 3... IDs únicos
