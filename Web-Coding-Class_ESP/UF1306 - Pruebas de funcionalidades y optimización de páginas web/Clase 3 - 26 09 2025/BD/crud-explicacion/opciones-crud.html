Opciones CRUD: SQL, HTML & CSS

Create

Inserta un nuevo producto en la tabla productos:

-- SQL INSERT INTO productos (nombre, precio) VALUES ('Camisa', 19.99);

Read

Recupera todos los productos:

-- SQL SELECT id, nombre, precio FROM productos;
ID Nombre Precio (€)
1 Camisa 19.99
2 Pantalón 29.99

Update

Modifica el precio de un producto existente (ID = 1):

-- SQL UPDATE productos SET precio = 24.99 WHERE id = 1;

Delete

Elimina un producto por su id (ID = 2):

-- SQL DELETE FROM productos WHERE id = 2;
