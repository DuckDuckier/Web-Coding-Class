¿Qué es XAMPP?

XAMPP es un programa gratuito que convierte tu ordenador en un servidor local. Esto te permite crear y probar páginas web en tu PC sin necesidad de estar conectado a Internet.

¿Qué significa XAMPP?

Letra Significado Función
X Multiplataforma Funciona en Windows, Linux y Mac
A Apache Servidor web
M MariaDB/MySQL Base de datos
P PHP Lenguaje de programación web
P Perl Otro lenguaje útil para tareas web

Ejemplo práctico

Imagina que estás creando una página web para un proyecto escolar. Con XAMPP puedes:

  • Escribir tu código en HTML, CSS y PHP
  • Guardarlo en la carpeta htdocs de XAMPP
  • Abrir tu navegador y escribir http://localhost para ver tu página funcionando
