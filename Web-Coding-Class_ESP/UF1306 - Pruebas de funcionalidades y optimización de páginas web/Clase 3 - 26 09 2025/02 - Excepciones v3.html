Tipos de Excepciones en PHP + Ejemplos

¿Qué son las excepciones en PHP?

Una excepción es una forma de detectar y manejar errores en tiempo de ejecución. En lugar de que el programa se detenga abruptamente, PHP puede “lanzar” una excepción y tú puedes “capturarla” para decidir cómo actuar.

PHP usa las palabras clave try, catch, throw y finally para trabajar con excepciones. Aprender a usarlas mejora la calidad del código y prepara a los alumnos para trabajar con bases de datos, APIs y sistemas reales.

Tipos comunes de excepciones en PHP

  • Exception: Clase base para todas las excepciones estándar. Se usa para errores generales en la lógica del programa.
  • Error: Representa errores graves del sistema, como problemas de sintaxis o memoria.
  • TypeError: Se lanza cuando se pasa un tipo de dato incorrecto a una función con tipado estricto.
  • DivisionByZeroError: Se lanza automáticamente cuando se intenta dividir por cero.
  • mysqli_sql_exception: Excepción específica de MySQLi que se lanza cuando ocurre un error en una consulta SQL.
Clase Descripción Ejemplo
Exception Clase base para todas las excepciones estándar 
try {
  throw new Exception("Algo salió mal");
} catch (Exception $e) {
  echo "Error: " . $e->getMessage();
}
Se lanza una excepción genérica y se captura con catch.
Error Errores graves del sistema 
try {
  eval('echo "Hola";'); // Código válido
  eval('echo Hola');    // Error de sintaxis
} catch (Error $e) {
  echo "Error de sistema: " . $e->getMessage();
}
Se captura un error de sintaxis con Error.
TypeError Tipo de dato incorrecto en una función 
function sumar(int $a, int $b) {
  return $a + $b;
}

try {
  echo sumar("uno", 2); // Error: "uno" no es entero
} catch (TypeError $e) {
  echo "Error de tipo: " . $e->getMessage();
}
Se lanza un error por pasar una cadena donde se espera un entero.
DivisionByZeroError División por cero 
try {
  $resultado = 10 / 0;
} catch (DivisionByZeroError $e) {
  echo "No puedes dividir por cero: " . $e->getMessage();
}
PHP lanza esta excepción automáticamente al dividir por cero.
mysqli_sql_exception Error en consulta SQL con MySQLi 
mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
$conexion = new mysqli("localhost", "root", "", "contacto");

try {
  $conexion->query("INSERT INTO mensajess (nombre) VALUES ('Juan')");
} catch (mysqli_sql_exception $e) {
  echo "Error SQL: " . $e->getMessage();
}
Se lanza una excepción porque la tabla mensajess no existe.
Throwable Captura cualquier excepción o error 
try {
  throw new Exception("Excepción genérica");
} catch (Throwable $e) {
  echo "Capturado con Throwable: " . $e->getMessage();
}
Útil para capturar cualquier tipo de error o excepción.
