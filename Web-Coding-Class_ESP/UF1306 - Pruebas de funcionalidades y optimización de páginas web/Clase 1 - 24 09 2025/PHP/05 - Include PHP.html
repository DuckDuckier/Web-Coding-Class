Funciones include() en PHP

Función Descripción Ejemplo
include() Inserta el contenido de otro archivo. Si no existe, muestra un warning pero continúa. include("menu.php");
require() Similar a include, pero si el archivo no existe, detiene el script con un error fatal. require("config.php");
include_once() Incluye el archivo solo una vez, incluso si se llama varias veces. include_once("header.php");
require_once() Igual que require, pero evita incluir el archivo más de una vez. require_once("funciones.php");
