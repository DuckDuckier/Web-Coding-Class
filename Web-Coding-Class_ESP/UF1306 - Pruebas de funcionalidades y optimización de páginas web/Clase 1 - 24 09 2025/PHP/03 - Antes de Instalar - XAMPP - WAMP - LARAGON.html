¿Qué tener en cuenta antes de instalar?

XAMPP, WAMP o Laragon

Sistema operativo compatible

  • XAMPP: funciona en Windows, macOS y Linux.
  • WAMP y Laragon: solo funcionan en Windows.

Permisos de administrador

  • Ejecuta el instalador como administrador para evitar errores.
  • Algunos servicios necesitan permisos elevados para funcionar correctamente.

Conflictos de puertos

  • Apache usa el puerto 80, MySQL el 3306.
  • Programas como Skype o IIS pueden bloquear esos puertos.
  • Solución: cambiar los puertos o cerrar los servicios que los usan.

Ruta de instalación

  • Instala en una ruta simple como C:\xampp o C:\laragon.
  • Evita carpetas con espacios o caracteres especiales.

Versiones de PHP y MySQL

  • Algunos proyectos requieren versiones específicas.
  • Laragon permite cambiar versiones fácilmente.

Compatibilidad con frameworks

  • ¿Usarás Laravel, WordPress u otros?
  • Laragon está optimizado para Laravel y Composer.

Antivirus y firewall

  • Podrían bloquear Apache o MySQL.
  • Permite el acceso en el firewall y añade excepciones en el antivirus.

Tu objetivo como desarrollador

  • XAMPP: ideal para empezar y usar en cualquier sistema.
  • WAMP: clásico y estable para Windows.
  • Laragon: moderno, rápido y flexible para proyectos avanzados.
