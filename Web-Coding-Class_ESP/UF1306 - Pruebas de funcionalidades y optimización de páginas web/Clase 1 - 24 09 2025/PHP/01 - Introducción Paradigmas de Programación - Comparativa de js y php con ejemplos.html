Paradigmas de Programación con Ejemplos

Imperativo

Es como seguir una receta paso a paso. Tú le dices al ordenador qué hacer y en qué orden.

Ejemplo: “Corta las verduras, calienta el aceite, fríe las verduras.”
JavaScript
let verduras = ["zanahoria", "cebolla"]; console.log("Cortar las verduras"); console.log("Calentar el aceite"); console.log("Freír las verduras");
PHP
$verduras = ["zanahoria", "cebolla"]; echo "Cortar las verduras\n"; echo "Calentar el aceite\n"; echo "Freír las verduras\n";

Declarativo

Solo dices lo que quieres lograr, sin explicar cómo hacerlo. El sistema decide los pasos.

Ejemplo: “Quiero ir al parque.” (Google Maps decide la ruta)
JavaScript
let numeros = [1, 2, 3, 4, 5]; let pares = numeros.filter(n => n % 2 === 0); console.log(pares); // [2, 4]
PHP
$numeros = [1, 2, 3, 4, 5]; $pares = array_filter($numeros, fn($n) => $n % 2 === 0); print_r($pares); // [1 => 2, 3 => 4]

Orientado a Objetos

Todo se organiza en objetos con propiedades y acciones. Como un coche que puede arrancar o frenar.

Ejemplo: “Crea un coche con color rojo que pueda acelerar.”
JavaScript
class Coche { constructor(color) { this.color = color; } acelerar() { console.log("El coche está acelerando"); } } let miCoche = new Coche("rojo"); miCoche.acelerar();
PHP
class Coche { public $color; public function __construct($color) { $this->color = $color; } public function acelerar() { echo "El coche está acelerando\n"; } } $miCoche = new Coche("rojo"); $miCoche->acelerar();

Funcional

Usa funciones que transforman datos sin cambiar nada fuera de ellas. Como una calculadora.

Ejemplo: “Suma 2 + 3 y siempre obtén 5.”
JavaScript
function sumar(a, b) { return a + b; } console.log(sumar(2, 3)); // 5
PHP
function sumar($a, $b) { return $a + $b; } echo sumar(2, 3); // 5

Lógico

Se basa en reglas y deducciones. Como un detective que resuelve un caso con pistas.

Ejemplo: “Si llueve, la calle está mojada. La calle está mojada, entonces ha llovido.”
JavaScript
let llueve = true; let calleMojada = llueve ? true : false; if (calleMojada) { console.log("Ha llovido"); }
PHP
$llueve = true; $calleMojada = $llueve ? true : false; if ($calleMojada) { echo "Ha llovido\n"; }
