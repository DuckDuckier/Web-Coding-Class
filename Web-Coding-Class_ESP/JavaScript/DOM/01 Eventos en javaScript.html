Eventos en JavaScript

Responde a las acciones del usuario en tu sitio web

¿Qué son los eventos?

Los eventos son acciones que ocurren en el navegador, como hacer clic, mover el mouse o presionar una tecla. JavaScript permite reaccionar a estos eventos mediante funciones llamadas controladores de eventos. El método más común para trabajar con ellos es addEventListener.

Ejemplo: Evento de clic en un botón

<!DOCTYPE html>
<html>
<head>
  <title>Eventos en JavaScript</title>
</head>
<body>
  <button id="myButton">Haz clic aquí</button>

  <script>
    // Obtener una referencia al botón
    const button = document.getElementById("myButton");

    // Adjuntar un controlador de eventos
    button.addEventListener("click", function() {
      alert("¡Has hecho clic en el botón!");
    });
  </script>
</body>
</html>
Este ejemplo muestra cómo detectar un clic en un botón y ejecutar una función que muestra un mensaje. Es la base para crear interacciones dinámicas en cualquier aplicación web.

Eventos comunes en JavaScript

element.addEventListener("click", ...) // clic del mouse
element.addEventListener("mouseenter", ...) // el mouse entra en el elemento
element.addEventListener("keydown", ...) // se presiona una tecla
element.addEventListener("submit", ...) // se envía un formulario
element.addEventListener("resize", ...) // se cambia el tamaño de la ventana
element.addEventListener("load", ...) // se carga la página
Puedes usar distintos tipos de eventos según la interacción que desees capturar. Consulta la documentación oficial para ver todos los eventos disponibles y sus propiedades.
