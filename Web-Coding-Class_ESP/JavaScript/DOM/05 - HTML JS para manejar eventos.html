Agregar eventos a elementos

Usa addEventListener para responder a acciones del usuario

Ejemplo: Evento click en un botón

HTML + JavaScript para manejar eventos con addEventListener

<!DOCTYPE html>
<html>
<head>
  <title>Agregar eventos a elementos en JavaScript</title>
</head>
<body>
  <!-- Botón al que se le agregará un evento -->
  <button id="myButton">Haz clic aquí</button>

  <script>
    // 1. Obtener una referencia al botón usando su ID
    const button = document.getElementById('myButton');

    // 2. Agregar un evento 'click' al botón
    button.addEventListener('click', function() {
      // 3. Código que se ejecuta cuando se hace clic
      console.log('Se hizo clic en el botón');
    });
  </script>
</body>
</html>
Este ejemplo muestra cómo usar addEventListener para manejar eventos:
  • getElementById: accede al botón por su ID.
  • addEventListener: adjunta el evento click al botón.
  • La función anónima se ejecuta cuando el usuario hace clic.
Puedes usar este método para cualquier tipo de evento: mouseover, keydown, submit, etc. También puedes usar funciones externas en lugar de funciones anónimas para mantener tu código organizado.
