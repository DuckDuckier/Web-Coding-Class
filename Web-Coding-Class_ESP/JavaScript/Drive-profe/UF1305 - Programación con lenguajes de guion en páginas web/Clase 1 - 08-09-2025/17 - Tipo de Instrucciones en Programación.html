Tipos de Instrucciones en Programación

Ejemplos interactivos con explicación

📌 Ejemplos por tipo de instrucción

let edad;

Se declara una variable llamada edad sin asignarle valor aún.

let suma = 5 + 3;

Se realiza una operación aritmética básica: suma de dos números.

edad = 25;

Se asigna el valor 25 a la variable edad.

let nombre = prompt("¿Cuál es tu nombre?");

Captura un dato introducido por el usuario mediante una ventana emergente.

console.log("Hola " + nombre);

Muestra un mensaje en la consola con el nombre capturado.

{
  let x = 10;
  let y = 20;
  console.log(x + y);
}

Bloque de instrucciones agrupadas entre llaves.

if (edad >= 18) {
  console.log("Eres mayor de edad");
}

Controla el flujo del programa según una condición.

if (edad >= 18) {
  console.log("Adulto");
} else {
  console.log("Menor");
}

Ejecuta una acción u otra según se cumpla la condición.

switch (dia) {
  case "lunes":
    console.log("Inicio de semana");
    break;
}

Salta a una sección específica del código si se cumple una condición.

for (let i = 0; i < 5; i++) {
  if (i === 3) continue;
  console.log(i);
}

Salta una iteración sin evaluar condición adicional (aquí con continue).

for (let i = 0; i < 3; i++) {
  console.log("Repetición " + i);
}

Ejecuta una instrucción varias veces mediante un bucle.
