Instrucciones de Salto en Programación

Condicionales e incondicionales con ejemplos interactivos

¿Qué son las instrucciones de salto?

Son instrucciones que permiten interrumpir el flujo secuencial de un programa y redirigirlo a otra parte del código. Se dividen en dos tipos: condicionales e incondicionales.

Salto condicional

Definición: El salto condicional se ejecuta solo si se cumple una condición. Es útil para tomar decisiones dentro del programa.

let edad = 20;

if (edad >= 18) {
  console.log("Eres mayor de edad");
}

En este ejemplo, el bloque console.log(...) solo se ejecuta si edad >= 18 es verdadero.

Se usa en estructuras como if, switch, while, for, etc.

Salto incondicional

Definición: El salto incondicional se ejecuta siempre, sin evaluar ninguna condición. Se usa para alterar el flujo de ejecución directamente.

for (let i = 0; i < 5; i++) {
  if (i === 3) continue;
  console.log(i);
}

En este ejemplo, cuando i === 3, se salta esa iteración sin ejecutar console.log(i).

Se usa con instrucciones como break, continue, return o goto (en otros lenguajes).
