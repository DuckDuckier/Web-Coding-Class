¿Qué es un algoritmo?

Explicación clara y tipos de algoritmos en programación

Definición

Un algoritmo es un conjunto de pasos ordenados y finitos que permiten resolver un problema o realizar una tarea específica. Es como una receta que indica qué hacer, en qué orden y cómo hacerlo.

Características de un buen algoritmo

  • Precisión: cada paso debe estar claramente definido.
  • Finitud: debe tener un número limitado de pasos.
  • Entrada: debe aceptar datos para procesar.
  • Salida: debe producir al menos un resultado.
  • Eficiencia: debe usar los recursos de forma óptima.

Tipos de algoritmos en programación

  • Algoritmos de búsqueda: encuentran un elemento en una estructura de datos.
  • Algoritmos de ordenamiento: organizan datos (por ejemplo, ordenar nombres alfabéticamente).
  • Algoritmos recursivos: se llaman a sí mismos para resolver subproblemas.
  • Algoritmos de grafos: trabajan con nodos y conexiones (como rutas o redes).
  • Algoritmos de cifrado: protegen datos mediante transformación y claves.

Ejemplo: Algoritmo para calcular el área de un triángulo


function calcularArea(base, altura) {
  return (base * altura) / 2;
}

let resultado = calcularArea(10, 5);
console.log("Área del triángulo: " + resultado);

Este algoritmo recibe dos entradas (base y altura), realiza un cálculo y devuelve una salida (el área).
