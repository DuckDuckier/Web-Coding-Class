Definición de A y B

Las variables A y B representan valores binarios en operaciones lógicas. En programación, se usan como booleanos:

  • 0 → falso (false)
  • 1 → verdadero (true)

Se combinan con operadores lógicos para tomar decisiones dentro de un programa.

¿Dónde se usan?

  • En estructuras de control como if, while, switch
  • En algoritmos de decisión
  • En validaciones de formularios
  • En sensores y sistemas electrónicos
  • En puertas lógicas digitales

Ejemplo práctico en JavaScript


let a = 1; // verdadero
let b = 0; // falso

if (a || b) {
  console.log("Al menos una condición es verdadera");
}

En este ejemplo, a y b se combinan con el operador OR (||) para evaluar si alguna condición es verdadera.
