▶️ Partes de un programa

Un programa se compone de partes que cumplen funciones específicas:

ParteDescripción
DeclaracionesDefinición de variables, constantes y estructuras de datos.
InstruccionesAcciones que el programa ejecuta.
Entrada de datosCaptura de información del usuario o sistema.
Proceso y salidaTransformación de datos y presentación del resultado.

🔍 Tipos de instrucciones

TipoDescripción
DeclaraciónDefine variables y constantes.
PrimitivaOperaciones básicas como suma, resta, etc.
AsignaciónAsigna valores a variables.
EntradaCaptura datos del usuario.
SalidaMuestra resultados.
CompuestaAgrupa instrucciones en bloques.
De controlDirige el flujo del programa.
AlternativaEjecuta según condiciones.
Salto condicionalCambia el flujo si se cumple una condición.
Salto incondicionalCambia el flujo sin condición.
RepetitivaEjecuta instrucciones varias veces.

🧩 Ejemplo práctico

Este programa en JavaScript lee dos números, los suma y muestra el resultado.


let a = parseInt(prompt("Introduce A"));
let b = parseInt(prompt("Introduce B"));
let resultado = a + b;
console.log("El resultado es: " + resultado);

Explicación:

  • Declaración: Se definen las variables a, b y resultado.
  • Entrada: Se capturan los valores mediante prompt().
  • Proceso: Se realiza la suma a + b.
  • Salida: Se muestra el resultado con console.log().
