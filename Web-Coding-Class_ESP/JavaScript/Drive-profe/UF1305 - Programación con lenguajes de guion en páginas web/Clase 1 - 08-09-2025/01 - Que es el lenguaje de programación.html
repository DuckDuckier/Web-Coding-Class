Lenguajes de Programación y sus Niveles

¿Qué es un lenguaje de programación?

Es un sistema que permite a los humanos comunicarse con las máquinas para crear software, páginas web, apps y más.

Ejemplo: console.log("Hola, mundo");

Niveles de Lenguajes

Nivel Nombre Ejemplo Características
1G Lenguaje máquina 01010101 Binario, directo al hardware
2G Ensamblador MOV AX, 5 Simbólico, específico del procesador
3G Alto nivel let x = 5; Legible, multiplataforma
4G Cuarta generación SELECT * FROM clientes Declarativo, orientado a tareas
5G Natural "Hazme una app de recetas" Interpretación de lenguaje humano

Ejemplo en JavaScript


let nombre = "Joaquín";
console.log("Hola, " + nombre);

Este código declara una variable y muestra un saludo en la consola.
