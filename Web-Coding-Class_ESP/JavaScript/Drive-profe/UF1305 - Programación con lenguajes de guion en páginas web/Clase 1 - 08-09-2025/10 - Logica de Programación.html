¿Qué es la lógica de programación?

Es la capacidad de estructurar ideas y pasos de forma coherente para resolver un problema mediante instrucciones que una computadora pueda ejecutar.

Permite pensar como programadores: dividir problemas, establecer condiciones, repetir acciones y tomar decisiones.

Aspectos clave a tener en cuenta

  • Pensamiento secuencial: instrucciones en orden lógico.
  • Condicionales: decisiones según condiciones.
  • Bucles: repetir acciones mientras se cumpla una condición.
  • Variables y operadores: almacenar y manipular datos.
  • Modularidad: dividir el código en funciones reutilizables.
  • Algoritmos: secuencia de pasos para resolver un problema.
