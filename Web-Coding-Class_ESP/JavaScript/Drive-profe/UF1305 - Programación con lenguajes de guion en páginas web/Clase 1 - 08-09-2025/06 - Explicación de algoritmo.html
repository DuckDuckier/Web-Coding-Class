Algoritmo de búsqueda

function buscarElemento(array, valor) {
Define una función que recibe un array y un valor a buscar.
for (let i = 0; i < array.length; i++) {
Recorre el array desde el índice 0 hasta el final.
if (array[i] === valor) return i;
Si el elemento actual es igual al valor buscado, devuelve su posición.
}
return -1;
Si no se encuentra el valor, devuelve -1.
}
console.log(buscarElemento(["Ana", "Luis", "Joaquín"], "Joaquín"));
Llama a la función y muestra la posición del nombre "Joaquín".

Algoritmo de ordenamiento

let numeros = [5, 2, 9, 1];
Declara un array con números desordenados.
numeros.sort((a, b) => a - b);
Ordena los números de menor a mayor usando una función de comparación.
console.log(numeros);
Muestra el array ordenado en consola.

Algoritmo recursivo

function factorial(n) {
Define una función para calcular el factorial de un número.
if (n === 0) return 1;
Caso base: el factorial de 0 es 1.
return n * factorial(n - 1);
Llamada recursiva: multiplica n por el factorial de n-1.
}
console.log(factorial(5));
Calcula el factorial de 5 y lo muestra en consola.
Que es el factorial de un número
El factorial de un número es una operación matemática que consiste en multiplicar ese número por todos los enteros positivos menores que él, hasta llegar al 1.
Se representa con un signo de exclamación: n! = n × (n - 1) × (n - 2) × ... × 2 × 1
Ejemplos básicos
4! = 4 × 3 × 2 × 1 = 24
6! = 6 × 5 × 4 × 3 × 2 × 1 = 720
0! = 1 (por convenio matemático)
