Tipos de Ordinograma

Clasificación según el tipo de flujo lógico que representan

📘 Ordinograma de Flujo

Representa el flujo secuencial de un programa. Muestra los pasos que se ejecutan uno tras otro, sin decisiones ni repeticiones. Es ideal para algoritmos lineales y simples.

🔀 Ordinograma de Decisión

Representa la lógica condicional de un programa. Incluye símbolos de decisión (rombos) que permiten bifurcar el flujo según condiciones. Se usa para algoritmos que requieren tomar decisiones.

🔁 Ordinograma de Bucle

Representa la repetición de procesos. Utiliza estructuras de bucle como while, for o do...while. Es útil para algoritmos que repiten acciones hasta cumplir una condición.
