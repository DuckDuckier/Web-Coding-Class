Clase e Instancia en Programación Orientada a Objetos

¿Qué es una clase?

Una clase es como un molde. Define cómo serán los objetos: qué atributos tendrán y qué funciones podrán ejecutar.

¿Qué es una instancia?

Una instancia es un objeto creado a partir de una clase. Es un ejemplo concreto con valores reales.

Ejemplo: Clase Persona

La clase Persona tiene atributos como nombre y edad, y un método llamado saludar().
