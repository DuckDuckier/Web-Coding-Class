¿Qué es una instancia?

Clase

Una clase es como un molde o plantilla que define cómo serán los objetos. Por ejemplo, una clase Persona puede tener atributos como nombre y edad, y métodos como saludar().

Instancia

Una instancia es un objeto concreto creado a partir de una clase. Es como hacer un pastel usando una receta: cada pastel es una instancia de la receta.
