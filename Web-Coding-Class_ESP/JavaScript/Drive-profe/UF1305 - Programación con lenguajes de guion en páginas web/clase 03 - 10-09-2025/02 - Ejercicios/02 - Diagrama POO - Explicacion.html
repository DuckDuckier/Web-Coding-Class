Programación orientada a objetos

En la programación orientada a objetos, usamos clases como moldes para crear objetos. Cada objeto tiene propiedades (atributos) y funciones (métodos) que definen su comportamiento.

↓
Clase

Clase: Es una plantilla que define cómo serán los objetos. Por ejemplo, una clase Persona puede tener nombre, edad y un método para saludar.

↓
Objeto
Objeto
Objeto

Objeto: Es una instancia de la clase. Por ejemplo, persona1 puede ser "Lucía, 20 años", creada a partir de la clase Persona.
