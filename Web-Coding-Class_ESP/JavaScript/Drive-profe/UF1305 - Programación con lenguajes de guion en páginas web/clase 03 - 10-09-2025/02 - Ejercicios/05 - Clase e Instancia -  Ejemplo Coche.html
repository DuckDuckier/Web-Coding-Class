Clase e Instancia en Programación Orientada a Objetos

Explicación sencilla

Imagina que tienes una clase llamada Coche. Esa clase define qué atributos tiene un coche (marca, modelo, color) y qué puede hacer (arrancar, frenar).

Cuando creas un coche específico —por ejemplo, un Toyota Corolla azul— estás creando una instancia de la clase Coche.

Analogía rápida

Concepto Ejemplo real
Clase El plano de diseño de un coche: define que todos los coches tienen ruedas, motor, color, etc.
Instancia Un coche específico que ya existe: un Toyota Corolla azul con matrícula 1234ABC.
