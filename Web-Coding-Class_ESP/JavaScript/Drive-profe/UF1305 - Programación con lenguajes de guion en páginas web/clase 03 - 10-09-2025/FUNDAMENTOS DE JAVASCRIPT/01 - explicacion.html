Fundamentos de JavaScript

¿Qué es una variable?

Una variable es como una caja donde guardamos información para usarla más tarde. Por ejemplo, si quieres guardar tu nombre:

let nombre = "Lucía"; console.log(nombre); // muestra "Lucía"

Usamos let para crear variables que pueden cambiar.

¿Qué es una constante?

Una constante es como una caja que no se puede abrir ni cambiar. Lo que guardas ahí, se queda igual:

const PI = 3.1416; console.log(PI); // muestra 3.1416

Usamos const para valores que no deben cambiar.

¿Qué es un array?

Un array (o arreglo) es una lista ordenada de valores. Es útil cuando queremos guardar varios datos relacionados.

Por ejemplo, si queremos guardar los nombres de varios alumnos:

let alumnos = ["Lucía", "Carlos", "María"]; console.log(alumnos[0]); // muestra "Lucía" console.log(alumnos.length); // muestra 3 (cantidad de elementos)

Los elementos del array se guardan en posiciones llamadas índices, que empiezan desde 0.

Podemos acceder a cada elemento usando su número de posición:

console.log(alumnos[1]); // muestra "Carlos"

También podemos agregar o quitar elementos:

alumnos.push("Sofía"); // añade "Sofía" al final alumnos.pop(); // elimina el último elemento

Los arrays pueden contener cualquier tipo de dato: textos, números, booleanos, objetos e incluso otros arrays.

let mezcla = ["Hola", 42, true, { nombre: "Ana" }, [1, 2]]; console.log(mezcla[3].nombre); // muestra "Ana"

¿Qué es un objeto?

Un objeto es una forma de agrupar información relacionada. Es como una ficha que describe algo con sus características.

Por ejemplo, si queremos guardar información sobre una persona:

let persona = { nombre: "Ana", edad: 12, esEstudiante: true }; console.log(persona.nombre); // muestra "Ana" console.log(persona.edad); // muestra 12

Un objeto tiene propiedades (como nombre o edad) y cada propiedad tiene un valor.

También puede tener funciones, llamadas métodos, que hacen cosas con el objeto.

let coche = { marca: "Toyota", modelo: "Corolla", color: "Azul", describir: function() { return "Este coche es un " + this.marca + " " + this.modelo + " de color " + this.color + "."; } }; console.log(coche.describir()); // muestra la descripción del coche

Los objetos son muy útiles para representar cosas del mundo real en nuestros programas.

Tipos de datos en JavaScript

JavaScript puede guardar diferentes tipos de información:

  • Texto (string): "Hola"
  • Número (number): 42
  • Booleano (boolean): true o false
  • Lista (array): [1, 2, 3]
  • Objeto (object): { nombre: "Ana", edad: 12 }
let edad = 12; // número let saludo = "Hola"; // texto let esAlumno = true; // booleano let notas = [8, 9, 10]; // array let persona = { nombre: "Ana", edad: 12 }; // objeto
