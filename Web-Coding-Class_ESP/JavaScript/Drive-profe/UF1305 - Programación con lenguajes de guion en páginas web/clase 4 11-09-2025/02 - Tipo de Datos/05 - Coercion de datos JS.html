Coerción de datos en JavaScript

Cómo se convierten los valores de un tipo a otro automáticamente o manualmente

La coerción de datos ocurre cuando JavaScript convierte un valor de un tipo a otro. Esto puede pasar automáticamente (implícita) o cuando lo hacemos nosotros (explícita).

Coerción implícita

JavaScript convierte el tipo de dato sin que lo pidamos. Esto puede ser útil, pero también causar errores si no lo entendemos bien.

// edad es un número
var edad = 30;

// JavaScript convierte edad a texto automáticamente
var mensaje = "Tengo " + edad + " años.";

console.log(mensaje); // "Tengo 30 años."

Coerción explícita

Aquí tú decides convertir el tipo de dato usando funciones como Number(), String() o Boolean().

// Convertimos texto a número usando Number()
var numeroComoCadena = "42";
var numeroComoNumero = Number(numeroComoCadena);

console.log(typeof numeroComoCadena); // "string"
console.log(typeof numeroComoNumero); // "number"

Convertir a texto (string)

// Usamos toString() para convertir número a texto
var numero = 42;
var texto = numero.toString();

console.log(typeof texto); // "string"

Convertir a número (number)

// Usamos Number() y parseInt() para convertir texto a número
var texto1 = "42";
var texto2 = "3.14";

var numero1 = Number(texto1);     // 42
var numero2 = parseInt(texto2);   // 3

console.log(typeof numero1); // "number"
console.log(typeof numero2); // "number"

Convertir a booleano (boolean)

Valores como 0, "", null, undefined y NaN se convierten en false. Todo lo demás es true.

// Convertimos diferentes valores a booleano
var valor1 = 1;
var valor2 = "";
var valor3 = undefined;
var valor4 = null;

console.log(Boolean(valor1)); // true
console.log(Boolean(valor2)); // false
console.log(Boolean(valor3)); // false
console.log(Boolean(valor4)); // false

Buenas prácticas

  • ✅ Usa === y !== en lugar de == y != para evitar errores por coerción automática.
  • ✅ Usa Number(), parseInt() y parseFloat() para convertir datos tú mismo.
  • ✅ Sé claro al convertir valores. Por ejemplo: parseInt("10", 10) es mejor que parseInt("10").
  • ✅ Recuerda que "" (cadena vacía) se convierte en 0 si se usa en operaciones matemáticas.
  • ✅ Usa isNaN() para verificar si un valor no es un número.
  • ✅ Usa typeof para saber qué tipo de dato tiene una variable.
Consejo: La coerción puede ayudarte, pero también confundirte. Sé explícito al convertir tipos de datos para evitar errores y escribir código más claro.
