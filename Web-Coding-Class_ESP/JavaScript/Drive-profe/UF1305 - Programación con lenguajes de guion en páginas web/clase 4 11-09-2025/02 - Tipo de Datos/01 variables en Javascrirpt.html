Variables en JavaScript

Aprende a guardar y usar información en tu código

Las variables son como cajas donde guardamos información que luego podemos usar en nuestro programa. En JavaScript, usamos tres formas para declarar variables: var, let y const.

Declaración de variables

Estas son las tres formas principales:

  • var: Se usa para declarar variables que pueden ser accedidas desde cualquier parte del código (menos recomendable hoy en día).
  • let: Se usa para declarar variables que solo existen dentro del bloque donde se crean (más seguro).
  • const: Se usa para declarar variables que no cambian su valor una vez asignado.

Ejemplos básicos

// Usando var
var saludo = "Hola mundo";

// Usando let
let edad = 25;

// Usando const
const pais = "España";

Ejemplo completo

var nombre = "Juan";
let ciudad = "Madrid";
const edad = 30;
var programador = true;
const id = 123456789;
let bandas = ["Queen", "The Beatles", "AC/DC"];

const persona = {
  nombre: nombre,
  ciudad: ciudad,
  edad: edad,
  programador: programador,
  id: id,
  bandas: bandas
};

console.log(persona);

Buenas prácticas

  • Usa nombres claros y descriptivos (ej. nombre en vez de n).
  • Evita usar palabras reservadas como function o class.
  • No uses espacios ni caracteres especiales en los nombres.
  • Los nombres no deben empezar con números.
  • Usa minúsculas y estilo camelCase para tus variables.
  • Si es una constante, puedes usar mayúsculas (ej. PI).
Consejo: Usa let y const en lugar de var. Son más seguros y modernos.
