Parámetros vs Variables en JavaScript

¿Qué es un parámetro?

Un parámetro es un identificador que se usa en la declaración de una función para recibir valores externos.

Ejemplo:

function saludar(nombre) { ... }

¿Qué es una variable?

Una variable es un espacio de memoria que almacena un valor. Puede existir dentro o fuera de funciones.

Ejemplo:

let edad = 25;
