Tipos de datos en JavaScript

Conoce cómo se guarda la información en tu código

En JavaScript, los tipos de datos definen qué tipo de información puede guardar una variable. Por ejemplo: texto, números, listas, objetos, etc.

Aprender esto es como conocer las piezas de un juego: cada tipo tiene su función y comportamiento.

1. Texto (string)

Se usa para guardar palabras, frases o cualquier texto.

let nombre = "Juan";
let saludo = "Hola, " + nombre; // Hola, Juan

También puedes usar saltos de línea (\n) o tabulaciones (\t):

let mensaje = "Línea 1\nLínea 2";
let titulo = "Inicio\tMi sitio web";

2. Números (number)

Se usa para guardar valores numéricos, ya sean enteros o decimales.

let edad = 30;
let precio = 19.99;

3. Verdadero o falso (boolean)

Solo tiene dos valores posibles: true (verdadero) o false (falso).

let esMayorDeEdad = true;
let tieneLicencia = false;

4. Sin valor definido (undefined)

Una variable que fue creada pero no tiene ningún valor asignado.

let resultado;
console.log(resultado); // undefined

5. Valor vacío (null)

Se usa para indicar que una variable no tiene ningún valor, de forma intencional.

let usuario = null;

6. Símbolo único (symbol)

Se usa para crear identificadores únicos. No se usa mucho al principio, pero es parte del lenguaje.

let id = Symbol("usuario");

7. Números grandes (bigint)

Se usa para guardar números muy grandes que no caben en number.

let numeroGrande = 123456789012345678901234567890n;

8. Listas (array)

Guarda varios valores en una sola variable, como una lista de frutas o números.

let frutas = ["manzana", "naranja", "plátano"];
let edades = [20, 25, 30];

9. Objetos (object)

Guarda información más compleja en forma de pares clave-valor.

let persona = {
  nombre: "Juan",
  edad: 30,
  tieneLicencia: true,
  bandasFavoritas: ["Queen", "The Beatles"]
};

¿Cómo saber qué tipo de dato tiene una variable?

Usa el operador typeof para averiguarlo:

let nombre = "Juan";
console.log(typeof nombre); // string

let edad = 30;
console.log(typeof edad); // number

let activo = true;
console.log(typeof activo); // boolean

let sinValor;
console.log(typeof sinValor); // undefined

let vacio = null;
console.log(typeof vacio); // object

let simbolo = Symbol("id");
console.log(typeof simbolo); // symbol

let grande = 123456789n;
console.log(typeof grande); // bigint

let frutas = ["manzana", "pera"];
console.log(typeof frutas); // object

let persona = { nombre: "Ana", edad: 25 };
console.log(typeof persona); // object
Consejo: Aprender los tipos de datos te ayuda a entender cómo funciona JavaScript y cómo manejar la información correctamente.
