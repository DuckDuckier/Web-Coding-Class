Operadores en JavaScript

Aprende a realizar operaciones y comparaciones en tu código

Los operadores son símbolos que permiten realizar cálculos, asignaciones, comparaciones y decisiones en JavaScript. Aquí te mostramos los más importantes con ejemplos fáciles de entender.

Operadores aritméticos

Se usan para hacer operaciones matemáticas básicas.

var x = 10;
var y = 5;

var suma = x + y;           // Suma
var resta = x - y;          // Resta
var multiplicacion = x * y; // Multiplicación
var division = x / y;       // División

console.log(suma);           // 15
console.log(resta);          // 5
console.log(multiplicacion); // 50
console.log(division);       // 2

Operadores de asignación

Sirven para guardar valores en variables y modificarlos.

var x = 10;

x += 5;  // x = x + 5
x -= 3;  // x = x - 3
x *= 2;  // x = x * 2
x /= 4;  // x = x / 4

console.log(x); // Resultado final: 6

Operadores de comparación

Comparan dos valores y devuelven true o false.

var x = 10;
var y = 5;

console.log(x > y);   // true
console.log(x < y);   // false
console.log(x >= y);  // true
console.log(x <= y);  // false
console.log(x == y);  // false (compara solo el valor)
console.log(x != y);  // true
console.log(5 === "5"); // false (compara valor y tipo)

Operador de igualdad estricta (===)

Compara si dos valores son iguales en tipo y valor. Es más seguro que ==.

console.log(5 === 5);           // true
console.log(5 === "5");         // false
console.log(true === 1);        // false
console.log(null === undefined);// false
console.log("hello" === "hello");// true
console.log({} === {});         // false (objetos distintos)

Operadores lógicos

Permiten combinar condiciones.

var x = 10;
var y = 5;

console.log(x > 5 && y < 10); // true (ambas condiciones son verdaderas)
console.log(x > 5 || y > 10); // true (una condición es verdadera)
console.log(!(x > y));        // false (niega la condición)

Operador ternario

Permite tomar decisiones rápidas en una sola línea.

var edad = 20;
var esMayorDeEdad = edad >= 18 ? "Sí" : "No";

console.log(esMayorDeEdad); // "Sí"

Buenas prácticas

  • ✅ Usa === y !== en lugar de == y != para evitar errores por conversión automática.
  • ✅ Evita comparar null y undefined con ===. Usa == si quieres verificar si una variable no tiene valor.
  • ✅ Usa operadores lógicos (&&, ||) en lugar de operadores bit a bit (&, |) para condiciones.
  • ✅ Evita usar ++ y -- si no es necesario. Prefiere x = x + 1 por claridad.
  • ✅ Usa paréntesis para controlar el orden de evaluación si tienes dudas.
  • ✅ Puedes usar condición && valor1 || valor2 como alternativa al operador ternario para simplificar.
Consejo: Los operadores son herramientas poderosas. Úsalos con claridad y cuidado para evitar errores y escribir código más limpio.
