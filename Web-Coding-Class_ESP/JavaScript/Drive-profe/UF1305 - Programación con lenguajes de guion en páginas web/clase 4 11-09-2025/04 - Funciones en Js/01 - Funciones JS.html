Funciones en JavaScript

Aprende a crear, usar y reutilizar bloques de código

Ejemplos de funciones

1. Función sin parámetros que imprime un mensaje.

function saludar() {
  console.log("Hola, bienvenido!");
}

saludar(); // Hola, bienvenido!
Se define una función llamada saludar sin parámetros. Dentro de la función, se imprime un mensaje en consola. Luego se llama a la función con saludar().

2. Función con un parámetro que personaliza el saludo.

function saludar(nombre) {
  console.log("Hola, " + nombre + "!");
}

saludar("Juan"); // Hola, Juan!
La función saludar recibe un parámetro nombre. Concatena ese valor con el mensaje y lo imprime. Se llama con "Juan" como argumento.

3. Función con dos parámetros para saludar con nombre y apellido.

function saludar(nombre, apellido) {
  console.log("Hola, " + nombre + " " + apellido + "!");
}

saludar("Juan", "Pérez"); // Hola, Juan Pérez!
La función recibe dos parámetros: nombre y apellido. Los concatena y muestra un saludo completo en consola.

4. Función que suma dos números y muestra el resultado.

function sumar(a, b) {
  var resultado = a + b;
  console.log("El resultado de la suma es: " + resultado);
}

sumar(5, 7); // El resultado de la suma es: 12
Se suman los valores a y b dentro de la función. El resultado se guarda en una variable y se imprime.

5. Función que devuelve un valor con return.

function sumar(a, b) {
  return a + b;
}

var resultado = sumar(5, 7);
console.log("El resultado de la suma es: " + resultado);
La función sumar devuelve la suma de a y b. El valor se guarda en resultado y se imprime.

6. Mostrar directamente el valor retornado en consola.

function sumar(a, b) {
  return a + b;
}

console.log("El resultado de la suma es: " + sumar(5, 7));
Se llama a sumar(5, 7) dentro de console.log(). El valor retornado se imprime directamente.

7. Función que multiplica dos números y devuelve el resultado.

function multiplicar(a, b) {
  return a * b;
}

var resultado = multiplicar(3, 4);
console.log("El resultado de la multiplicación es: " + resultado);
La función multiplicar devuelve el producto de a y b. El resultado se guarda en una variable y se imprime.

8. Función anónima como argumento de setTimeout.

setTimeout(function() {
  console.log("Han pasado 5 segundos");
}, 5000);
Se define una función anónima dentro de setTimeout. Después de 5 segundos, se ejecuta y muestra el mensaje en consola.
