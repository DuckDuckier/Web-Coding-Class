Ejercicios de funciones en JavaScript

Practica creando y utilizando funciones con lógica condicional y matemática

Funciones

Ejercicio 1: Imprimir los primeros “n” números pares.

function pares(n) {
  var contador = 0;
  var numero = 2;
  while (contador < n) {
    console.log(numero);
    numero += 2;
    contador++;
  }
}
pares(6);
La función imprime los primeros n números pares comenzando desde 2. Utiliza un bucle while para controlar la cantidad de pares mostrados.

Ejercicio 2: Mostrar nombre completo en un alert.

function nombreCompleto(nombre, apellido) {
  alert(nombre + " " + apellido);
}
nombreCompleto("Pedro", "Gonzalez");
La función concatena nombre y apellido y lo muestra en un alert().

Ejercicio 3: Determinar si puedes solicitar permiso de conducir. 

function permiso(nombre, edad) {
  if (edad >= 18) {
    alert(nombre + ", puedes solicitar el permiso");
  } else {
    alert(nombre + ", no puedes solicitar el permiso");
  }
}
permiso("Roberto", 15);
Se evalúa si la edad es mayor o igual a 18. Se muestra un mensaje personalizado según el resultado.

Ejercicio 4: Calcular promedio de 5 calificaciones y mostrar si aprobó. 

function promedio(c1, c2, c3, c4, c5) {
  var promedio = (c1 + c2 + c3 + c4 + c5) / 5;
  if (promedio >= 70) {
    alert("Aprobado: " + promedio);
  } else {
    alert("No aprobado: " + promedio);
  }
}
promedio(70, 80, 80, 90, 60);
Se calcula el promedio de 5 calificaciones. Si el promedio es mayor o igual a 70, se muestra "Aprobado". Si no, se muestra "No aprobado".

Ejercicio 8: Función que devuelve el factorial de un número.

function factorial(n) {
  let resultado = 1;
  for (let i = 1; i <= n; i++) {
    resultado *= i;
  }
  return resultado;
}
console.log(factorial(5)); // 120
Multiplica todos los números desde 1 hasta n para calcular el factorial.

Ejercicio 10: Verificar si un número es primo.

function esPrimo(n) {
  if (n < 2) return false;
  for (let i = 2; i <= Math.sqrt(n); i++) {
    if (n % i === 0) return false;
  }
  return true;
}
console.log(esPrimo(7)); // true
La función verifica si n es divisible por algún número entre 2 y su raíz cuadrada. Si lo es, no es primo. Si no, devuelve true.

Ejercicio 11: Devolver el número más grande de un array.

function maximo(arr) {
  return Math.max(...arr);
}
console.log(maximo([3, 8, 2, 9, 5])); // 9
Usa el operador spread ... para pasar los elementos del array a Math.max(). Devuelve el número más alto.

Ejercicio 12: Contar cuántas vocales tiene una cadena.

function contarVocales(texto) {
  return (texto.match(/[aeiou]/gi) || []).length;
}
console.log(contarVocales("Hola mundo")); // 4
Usa una expresión regular para buscar vocales en la cadena. match() devuelve un array con las coincidencias, y length cuenta cuántas hay.

Ejercicio 13: Invertir una cadena de texto.

function invertirCadena(texto) {
  return texto.split("").reverse().join("");
}
console.log(invertirCadena("JavaScript")); // tpircSavaJ
Convierte la cadena en un array con split(), lo invierte con reverse() y lo une con join().

Ejercicio 14: Generar un número aleatorio entre dos valores.

function aleatorio(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}
console.log(aleatorio(1, 10)); // número entre 1 y 10
Math.random() genera un número entre 0 y 1. Se escala al rango deseado y se redondea con Math.floor().

Ejercicio 15: Devolver un nombre aleatorio de una lista.

function nombreAleatorio(nombres) {
  const indice = Math.floor(Math.random() * nombres.length);
  return nombres[indice];
}
console.log(nombreAleatorio(["Ana", "Luis", "Carlos", "María"]));
Genera un índice aleatorio entre 0 y la longitud del array. Devuelve el nombre que está en esa posición.
