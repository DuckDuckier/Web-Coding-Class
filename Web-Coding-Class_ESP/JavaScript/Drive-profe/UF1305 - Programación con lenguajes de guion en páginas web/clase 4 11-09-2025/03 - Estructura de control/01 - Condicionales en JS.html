Condicionales en JavaScript

Los condicionales permiten ejecutar diferentes bloques de código según ciertas condiciones. Los más comunes son if, else if, else, switch y el operador ternario.

Ejemplo 1: if

Ejecuta un bloque si se cumple una condición.

var edad = 18;

if (edad >= 18) {
  console.log("Eres mayor de edad");
}

Ejemplo 2: if / else

Ejecuta un bloque si no se cumple la condición del if.

var dia = "domingo";

if (dia === "sábado" || dia === "domingo") {
  console.log("¡Es fin de semana!");
} else {
  console.log("¡A trabajar!");
}

Ejemplo 3: else if

Evalúa múltiples condiciones en orden.

var hora = 14;

if (hora < 12) {
  console.log("Buenos días");
} else if (hora < 20) {
  console.log("Buenas tardes");
} else {
  console.log("Buenas noches");
}

Ejemplo 4: Operador ternario

Forma abreviada de if en una sola línea.

var edad = 20;
var esMayorDeEdad = edad >= 18 ? "Sí" : "No";

console.log(esMayorDeEdad); // "Sí"

Ejemplo 5: switch

Ejecuta diferentes bloques según el valor de una variable.

var dia = 5;
var mensaje;

switch (dia) {
  case 1:
    mensaje = "Lunes";
    break;
  case 2:
    mensaje = "Martes";
    break;
  case 3:
    mensaje = "Miércoles";
    break;
  case 4:
    mensaje = "Jueves";
    break;
  case 5:
    mensaje = "Viernes";
    break;
  case 6:
    mensaje = "Sábado";
    break;
  case 7:
    mensaje = "Domingo";
    break;
  default:
    mensaje = "Día inválido";
}

console.log(mensaje); // "Viernes"
Buenas prácticas:
  • ✅ Evita anidar condicionales innecesariamente.
  • ✅ Usa === en lugar de == para evitar conversiones implícitas.
  • ✅ Utiliza llaves {} siempre, incluso para una sola línea.
  • ✅ Simplifica condiciones con && y || cuando sea posible.
  • ✅ Usa Boolean() para convertir valores en booleanos de forma clara.
  • ✅ Evita negaciones dobles como !!valor si no es necesario.
  • ✅ Usa nombres de variables descriptivos para mejorar la legibilidad.
  • ✅ Asegúrate de cubrir todos los casos posibles con else o default.
  • ✅ Evalúa primero las condiciones más específicas.
