Iterando arreglos (arrays) en JavaScript

1. Iterar con for

var numeros = [1, 2, 3, 4, 5];
var suma = 0;

for (var i = 0; i < numeros.length; i++) {
  suma += numeros[i];
}

console.log("La suma de los números es:", suma); // La suma de los números es: 15
El bucle for recorre el arreglo numeros usando índices. En cada iteración, se suma el valor actual al acumulador suma. Este método es clásico y ofrece control total sobre el índice.

2. Iterar con for...of

var nombres = ["Juan", "María", "Pedro", "Ana"];

for (var nombre of nombres) {
  console.log(nombre);
}
// Juan
// María
// Pedro
// Ana
for...of recorre directamente los valores del arreglo sin usar índices. Es más legible y moderno, ideal para mostrar o procesar elementos.

3. Iterar con forEach()

var nombres = ["Juan", "María", "Pedro", "Ana"];

nombres.forEach(function(nombre, indice) {
  console.log(indice, nombre);
});
// 0 Juan
// 1 María
// 2 Pedro
// 3 Ana
forEach() ejecuta una función para cada elemento del arreglo. Recibe el valor y el índice como parámetros. Ideal para recorrer arreglos de forma funcional y clara.

4. Iterar con map()

var numeros = [1, 2, 3, 4, 5];

var dobleNumeros = numeros.map(function(numero) {
  return numero * 2;
});

console.log(dobleNumeros); // [2, 4, 6, 8, 10]
console.log(numeros);      // [1, 2, 3, 4, 5]
map() aplica una función a cada elemento del arreglo y devuelve un nuevo arreglo con los resultados. El arreglo original no se modifica. Ideal para transformar datos sin mutar el estado original.

Iteración avanzada de arreglos

5. Iterar con filter()

var numeros = [1, 2, 3, 4, 5];

var pares = numeros.filter(function(numero) {
  return numero % 2 === 0;
});

console.log(pares); // [2, 4]
filter() crea un nuevo arreglo con los elementos que cumplen una condición. En este caso, selecciona los números pares. El arreglo original no se modifica.

6. Iterar con find()

var numeros = [1, 2, 3, 4, 5];

var primerPar = numeros.find(function(numero) {
  return numero % 2 === 0;
});

console.log(primerPar); // 2
find() devuelve el primer elemento que cumple la condición. A diferencia de filter(), no devuelve un arreglo, sino un único valor.

7. Iterar con reduce()

var numeros = [1, 2, 3, 4, 5];

var suma = numeros.reduce(function(acumulador, numero) {
  return acumulador + numero;
}, 0);

console.log(suma); // 15
reduce() acumula un valor a lo largo del arreglo. En este ejemplo, suma todos los elementos. Ideal para cálculos como totales, promedios o combinaciones.

8. Iterar con every()

var numeros = [2, 4, 6, 8, 10];

var todosPares = numeros.every(function(numero) {
  return numero % 2 === 0;
});

console.log(todosPares); // true
every() comprueba si todos los elementos cumplen la condición. Devuelve true solo si todos los elementos la cumplen.

9. Iterar con some()

var numeros = [1, 3, 5, 7, 9];

var algunPar = numeros.some(function(numero) {
  return numero % 2 === 0;
});

console.log(algunPar); // false
some() comprueba si al menos un elemento cumple la condición. Devuelve true si encuentra uno solo que la cumpla.

10. Buenas prácticas al iterar arreglos

- Prefiere métodos como map(), filter(), reduce(), etc. en lugar de bucles for.
- Usa Array.isArray() para verificar si una variable es un arreglo.
- Nombra tus variables de forma descriptiva y clara.
- Evita mutar directamente los elementos del arreglo dentro del bucle.
- Usa const si no vas a reasignar la variable del bucle.
- Considera usar desestructuración para acceder a elementos del arreglo.
Estas prácticas ayudan a escribir código más legible, seguro y eficiente. Reducen errores comunes y mejoran la mantenibilidad del proyecto.
