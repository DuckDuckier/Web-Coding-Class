Arrays en JavaScript

1. ¿Qué es un array? 

Un array es una estructura de datos que permite almacenar múltiples valores en una sola variable.
Cada elemento tiene un índice numérico, comenzando desde 0.
Los arrays pueden contener cualquier tipo de dato: números, cadenas, booleanos, objetos, incluso otros arrays.

2. Crear un array vacío 

var miArray = [];           // Usando corchetes
var otroArray = new Array(); // Usando el constructor
Ambas formas crean un array vacío. Se recomienda usar corchetes por simplicidad y claridad.

3. Crear un array con elementos 

var miArray = [1, 'Hola', [3], true];
var otroArray = new Array(1, 2, 3);
Los arrays pueden contener valores heterogéneos. El constructor también permite inicializar con elementos.

4. Declarar arrays con diferentes palabras clave 

var miArray = [1, 2, 3];
let miArray = [1, 2, 3];
const miArray = [1, 2, 3];
Usa const si no vas a reasignar el array. Aunque el contenido pueda cambiar, la referencia permanece.

5. Acceder a un elemento del array 

var miArray = [1, 2, 3];
console.log(miArray[0]); // Muestra 1
El índice 0 corresponde al primer elemento. Puedes acceder a cualquier posición usando corchetes.

6. Modificar un elemento del array 

var miArray = [1, 2, 3];
miArray[0] = 4;
console.log(miArray[0]); // Muestra 4
Puedes cambiar el valor de cualquier elemento accediendo por su índice y asignando un nuevo valor.

7. Buenas prácticas con arrays 

- Usa const si no vas a reasignar el array.
- Evita modificar arrays directamente; usa métodos como map(), filter(), reduce().
- Prefiere métodos como forEach() en lugar de bucles for.
- Nombra tus arrays de forma descriptiva.
- Usa Array.isArray() para verificar si una variable es un array.
- Evita operaciones costosas como reordenar o eliminar elementos innecesariamente.
Estas prácticas ayudan a mantener tu código limpio, eficiente y fácil de mantener.
