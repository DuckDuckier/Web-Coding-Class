Arrow Functions con Arreglos en JavaScript

Aprende a recorrer, transformar y filtrar arreglos usando funciones flecha de forma moderna y eficiente

Arrow Functions con Arreglos

1. Usar arrow function con for

const numeros = [1, 2, 3, 4, 5];

const multiplicarPor2 = (arr) => {
  for (let i = 0; i < arr.length; i++) {
    console.log(arr[i] * 2);
  }
}

multiplicarPor2(numeros);
Esta función flecha recorre el arreglo con un bucle for tradicional. En cada iteración, imprime el doble del número actual.

2. Usar arrow function con forEach()

const numeros = [1, 2, 3, 4, 5];

const sumar = (arr) => {
  let suma = 0;
  arr.forEach(num => suma += num);
  console.log(suma);
}

sumar(numeros);
forEach() ejecuta una función flecha para cada elemento. Se acumula la suma en una variable externa.

3. Usar arrow function con map()

const numeros = [1, 2, 3, 4, 5];

const doble = (arr) => {
  const nuevoArr = arr.map(num => num * 2);
  console.log(nuevoArr);
}

doble(numeros);
map() transforma cada elemento del arreglo sin modificar el original. Devuelve un nuevo arreglo con los valores multiplicados por 2.

4. Usar arrow function con filter()

const numeros = [1, 2, 3, 4, 5];

const mayoresA3 = (arr) => {
  const nuevoArr = arr.filter(num => num > 3);
  console.log(nuevoArr);
}

mayoresA3(numeros);
filter() devuelve un nuevo arreglo con los elementos que cumplen la condición. En este caso, los números mayores a 3.

5. Usar arrow function con find()

const numeros = [1, 3, 5, 4, 7, 9, 6];

const encontrarPar = (arr) => {
  const primerPar = arr.find(num => num % 2 === 0);
  console.log(primerPar);
}

encontrarPar(numeros);
find() devuelve el primer elemento que cumple la condición. Aquí se busca el primer número par en el arreglo.

6. Usar arrow function con reduce()

const numeros = [1, 2, 3, 4, 5];

const suma = (arr) => {
  const total = arr.reduce((acc, num) => acc + num, 0);
  console.log(total);
}

suma(numeros);
reduce() acumula un valor a lo largo del arreglo. En este caso, suma todos los elementos y devuelve el total.

7. Buenas prácticas

- Usa const para declarar funciones flecha.
- Prefiere métodos como map(), filter(), reduce() en lugar de bucles.
- Usa la sintaxis de una sola línea cuando sea posible.
- Evita mutar directamente los elementos del arreglo.
- Usa nombres descriptivos para tus funciones.
- Verifica que el valor sea un arreglo con Array.isArray().
- Usa desestructuración para acceder a elementos del arreglo.
Estas prácticas ayudan a escribir código más limpio, legible y seguro. Las funciones flecha son ideales para trabajar con arreglos de forma funcional.
