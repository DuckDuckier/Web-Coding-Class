Agregar y eliminar elementos de un arreglo

1. Agregar elementos con push() 

let miArreglo = ['manzana', 'banana', 'cereza'];
miArreglo.push('durazno');
console.log(miArreglo); // ['manzana', 'banana', 'cereza', 'durazno']

// Agregar múltiples elementos
miArreglo.push('fresa', 'uva');
console.log(miArreglo); // ['manzana', 'banana', 'cereza', 'durazno', 'fresa', 'uva']
push() añade elementos al final del arreglo. Devuelve la nueva longitud del arreglo.

2. Agregar elementos con unshift() 

let miArreglo = ['manzana', 'banana', 'cereza'];
miArreglo.unshift('durazno');
console.log(miArreglo); // ['durazno', 'manzana', 'banana', 'cereza']

// Agregar múltiples elementos
miArreglo.unshift('fresa', 'uva');
console.log(miArreglo); // ['fresa', 'uva', 'durazno', 'manzana', 'banana', 'cereza']
unshift() agrega elementos al inicio del arreglo. También devuelve la nueva longitud.

3. Agregar elementos con el operador spread 

let miArreglo = ['manzana', 'banana', 'cereza'];
let otrosFrutos = ['durazno', 'fresa', 'uva'];
miArreglo = [...miArreglo, ...otrosFrutos];
console.log(miArreglo); // ['manzana', 'banana', 'cereza', 'durazno', 'fresa', 'uva']
El operador ... permite expandir los elementos de otro arreglo. Ideal para combinar arreglos sin mutarlos.

4. Agregar elementos con splice() 

let numeros = [1, 2, 3, 4, 5];
numeros.splice(2, 0, 6, 7);
console.log(numeros); // [1, 2, 6, 7, 3, 4, 5]
splice() permite insertar elementos en una posición específica. El segundo argumento indica cuántos elementos eliminar (0 en este caso).

4.1 Eliminar elementos con splice() (ejemplo detallado) 

                let numeros = [1, 2, 3, 4, 5];
                numeros.splice(2, 2); // Elimina 2 elementos desde la posición 2
                console.log(numeros); // [1, 2, 5]
splice() permite eliminar elementos desde una posición específica del arreglo. En este ejemplo, el arreglo original es [1, 2, 3, 4, 5]. La llamada splice(2, 2) indica que:
  • 2 es la posición inicial (índice 2, que corresponde al número 3)
  • 2 es la cantidad de elementos a eliminar (el número 3 y el número 4)
Después de ejecutar splice(), el arreglo queda como [1, 2, 5], ya que los elementos en las posiciones 2 y 3 han sido eliminados. Si deseas guardar los elementos eliminados, puedes hacerlo así: 
let eliminados = numeros.splice(2, 2);
console.log(eliminados); // [3, 4]
Esto es útil si necesitas procesar o registrar los elementos que fueron retirados.

5. Eliminar elementos con pop() 

let miArreglo = ['manzana', 'banana', 'cereza'];
let frutaEliminada = miArreglo.pop();
console.log(miArreglo);       // ['manzana', 'banana']
console.log(frutaEliminada);  // 'cereza'
pop() elimina el último elemento y lo devuelve. Es útil para pilas (estructura LIFO).

6. Eliminar elementos con shift() 

let miArreglo = ['manzana', 'banana', 'cereza'];
let frutaEliminada = miArreglo.shift();
console.log(miArreglo);       // ['banana', 'cereza']
console.log(frutaEliminada);  // 'manzana'
shift() elimina el primer elemento y lo devuelve. Es útil para colas (estructura FIFO).

7. Eliminar elementos con splice() 

let numeros = [1, 2, 3, 4, 5];
let eliminados = numeros.splice(2, 2);
console.log(numeros);       // [1, 2, 5]
console.log(eliminados);    // [3, 4]
splice() elimina elementos desde una posición específica. Devuelve los elementos eliminados en un nuevo arreglo.

8. Buenas prácticas al modificar arreglos 

- Prefiere métodos que no mutan el arreglo original: map(), filter(), reduce(), etc.
- Usa push() y unshift() para agregar elementos de forma eficiente.
- Usa pop() y shift() para eliminar elementos sin complicaciones.
- Evita eliminar elementos del medio del arreglo si no es necesario.
- Usa Array.isArray() para verificar si una variable es un arreglo.
- Sé consciente del impacto de métodos como sort(), reverse() y splice().
Estas prácticas ayudan a mantener tu código más predecible, legible y menos propenso a errores.

Métodos que no mutan el arreglo original

9. Usar map() para transformar un arreglo

const numeros = [1, 2, 3, 4, 5];
const dobles = numeros.map(num => num * 2);
console.log(dobles);  // [2, 4, 6, 8, 10]
console.log(numeros); // [1, 2, 3, 4, 5]
map() crea un nuevo arreglo aplicando una función a cada elemento. El arreglo original numeros no se modifica. Ideal para transformar datos sin alterar la fuente.

10. Usar filter() para seleccionar elementos

const edades = [12, 18, 25, 8, 30];
const mayoresDeEdad = edades.filter(edad => edad >= 18);
console.log(mayoresDeEdad); // [18, 25, 30]
console.log(edades);        // [12, 18, 25, 8, 30]
filter() devuelve un nuevo arreglo con los elementos que cumplen una condición. No modifica el arreglo original. Muy útil para extraer subconjuntos de datos.

11. Usar reduce() para acumular valores

const precios = [10, 20, 30];
const total = precios.reduce((acumulador, precio) => acumulador + precio, 0);
console.log(total); // 60
console.log(precios); // [10, 20, 30]
reduce() toma una función acumuladora y un valor inicial. Recorre el arreglo y acumula un resultado (en este caso, la suma total). No modifica el arreglo original. Ideal para cálculos como totales, promedios o combinaciones.
