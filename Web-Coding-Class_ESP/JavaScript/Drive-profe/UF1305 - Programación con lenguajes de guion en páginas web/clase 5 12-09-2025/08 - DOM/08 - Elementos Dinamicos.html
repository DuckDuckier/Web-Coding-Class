Elementos dinámicos

Crea y manipula elementos HTML en tiempo real con JavaScript

Ejemplo: Crear un párrafo dinámicamente

HTML + JavaScript para generar contenido en tiempo de ejecución

<!DOCTYPE html>
<html>
<head>
  <title>Elementos Dinámicos</title>
</head>
<body>
  <h1>Elementos Dinámicos</h1>

  <!-- Botón que activa la creación del párrafo -->
  <button onclick="agregarParrafo()">Agregar Párrafo</button>

  <script>
    // Función que crea un nuevo elemento <p>
    function agregarParrafo() {
      // 1. Crear el elemento
      const nuevoParrafo = document.createElement('p');

      // 2. Asignar contenido textual
      nuevoParrafo.textContent = 'Este es un párrafo creado dinámicamente';

      // 3. Insertar el elemento en el documento
      document.body.appendChild(nuevoParrafo);
    }
  </script>
</body>
</html>
Este ejemplo muestra cómo crear elementos HTML en tiempo real:
  • createElement: genera un nuevo nodo HTML.
  • textContent: asigna el texto que se mostrará.
  • appendChild: inserta el elemento en el DOM.
Puedes extender esta técnica para crear listas, tarjetas, formularios, menús y cualquier estructura HTML que necesites. También puedes añadir atributos, clases, estilos y eventos para enriquecer la interacción.
