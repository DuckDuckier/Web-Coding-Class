Crear elementos en el DOM

Aprende a generar contenido dinámico con JavaScript

Ejemplo: Crear y agregar un elemento al DOM

HTML + JavaScript para crear un nuevo párrafo

<!DOCTYPE html>
<html>
<head>
  <title>Creación de elementos en JavaScript</title>
</head>
<body>
  <div id="container"></div>

  <script>
    // 1. Obtener una referencia al contenedor donde se insertará el nuevo elemento
    const container = document.getElementById('container');

    // 2. Crear un nuevo elemento HTML tipo <p>
    const newElement = document.createElement('p');

    // 3. Asignar contenido textual al nuevo elemento
    newElement.textContent = 'Este es un nuevo párrafo creado con JavaScript';

    // 4. Insertar el nuevo elemento dentro del contenedor
    container.appendChild(newElement);
  </script>
</body>
</html>
Este ejemplo muestra cómo crear un elemento HTML dinámicamente:
  • getElementById: accede al contenedor donde se insertará el nuevo elemento.
  • createElement: genera un nuevo nodo HTML (en este caso, un párrafo).
  • textContent: asigna el texto que se mostrará dentro del nuevo elemento.
  • appendChild: agrega el nuevo elemento como hijo del contenedor.
Puedes seguir manipulando el elemento antes de insertarlo: añadirle clases, atributos, estilos o eventos. Esta técnica es esencial para construir interfaces dinámicas y reactivas.
