Selección del DOM

Aprende a acceder a elementos HTML usando distintos métodos de JavaScript

Ejemplo: Seleccionar elementos con distintos métodos

HTML base para practicar la selección del DOM

<!DOCTYPE html>
<html>
<head>
  <title>Selección de elementos del DOM</title>
</head>
<body>
  <h1 id="titulo">Título</h1>
  <p class="parrafo">Lorem ipsum dolor sit amet.</p>
  <ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
  </ul>
  <div>
    <p>Texto 1</p>
    <p>Texto 2</p>
    <p>Texto 3</p>
  </div>

  <script>
    // Selecciona el elemento con ID "titulo"
    const titulo = document.getElementById('titulo');
    console.log(titulo); // Muestra el <h1> en consola

    // Selecciona todos los elementos con clase "parrafo"
    const parrafos = document.getElementsByClassName('parrafo');
    console.log(parrafos); // HTMLCollection con los <p> que tienen esa clase

    // Selecciona todos los elementos <li>
    const elementosLista = document.getElementsByTagName('li');
    console.log(elementosLista); // HTMLCollection con los <li>

    // Selecciona el primer <p> dentro de un <div>
    const texto1 = document.querySelector('div p');
    console.log(texto1); // Primer <p> dentro del <div>

    // Selecciona todos los elementos <p> del documento
    const todosLosParrafos = document.querySelectorAll('p');
    console.log(todosLosParrafos); // NodeList con todos los <p>
  </script>
</body>
</html>
Este ejemplo muestra cómo usar distintos métodos para seleccionar elementos del DOM:
  • getElementById: selecciona por ID único.
  • getElementsByClassName: selecciona por clase (HTMLCollection).
  • getElementsByTagName: selecciona por etiqueta (HTMLCollection).
  • querySelector: selecciona el primer elemento que coincide con un selector CSS.
  • querySelectorAll: selecciona todos los elementos que coinciden con un selector CSS (NodeList).
Todos los resultados se muestran en la consola del navegador usando console.log(). Estos métodos son fundamentales para acceder y manipular elementos HTML desde JavaScript.
