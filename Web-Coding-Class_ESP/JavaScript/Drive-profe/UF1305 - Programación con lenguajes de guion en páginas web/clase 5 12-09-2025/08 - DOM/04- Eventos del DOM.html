Eventos del DOM

Captura acciones del usuario y responde dinámicamente con JavaScript

Ejemplo: Manejo de eventos comunes

HTML + JavaScript con múltiples eventos del DOM

<!DOCTYPE html>
<html>
<head>
  <title>Eventos del DOM</title>
</head>
<body>
  <h1>Eventos del DOM</h1>

  <!-- Evento onchange: se activa al cambiar el valor y perder el foco -->
  <input type="text" id="inputText" onchange="mostrarMensaje()">

  <!-- Evento onclick: se activa al hacer clic -->
  <button onclick="cambiarColor()">Haz clic</button>

  <!-- Eventos onmouseover y onmouseout: al pasar y salir el mouse -->
  <p onmouseover="mostrarMensaje('¡Pasaste el mouse por encima!')" 
     onmouseout="mostrarMensaje('¡Mouse fuera!')">
     Pasa el mouse aquí
  </p>

  <!-- Evento onkeydown: se activa al presionar una tecla -->
  <input type="text" onkeydown="mostrarTecla(event)">

  <script>
    // Muestra un mensaje en consola, dependiendo del contexto
    function mostrarMensaje(mensaje) {
      if (mensaje) {
        console.log(mensaje);
      } else {
        console.log("El valor del input ha cambiado");
      }
    }

    // Cambia el color de fondo del body al hacer clic
    function cambiarColor() {
      document.body.style.backgroundColor = "red";
    }

    // Muestra la tecla presionada en consola
    function mostrarTecla(event) {
      console.log("Tecla presionada: " + event.key);
    }
  </script>
</body>
</html>
Este ejemplo incluye varios eventos del DOM:
  • onchange: detecta cambios en un campo de texto.
  • onclick: responde al clic en un botón.
  • onmouseover / onmouseout: reaccionan al movimiento del mouse sobre un párrafo.
  • onkeydown: captura teclas presionadas en un campo de entrada.
Cada evento ejecuta una función que muestra información en la consola o modifica el estilo de la página. Puedes personalizar estas funciones para realizar validaciones, animaciones, actualizaciones dinámicas y más.
