10 Ejercicios de Arrays

funciones sin flechas y con flechas

Recordar que en progración, existe varias formas de resolver un problema, el codigo debe ser eficiente y facil de leer, pon siempre comentarios para comprender mejor

Resuelve los ejercicios sin flechas y con flechas

práctica para comprender mejor las dos formas de como se escribe el codigo tanto de flechas como sin flechas.

1. Calcular promedio y verificar aprobación

Crea una función que reciba un array de calificaciones, calcule el promedio y determine si el resultado es aprobado (mayor a 70).

2. Imprimir cada elemento de un array

Escribe una función que recorra un array y muestre cada elemento en la consola.

3. Obtener solo las calificaciones aprobadas

Filtra un array de calificaciones y devuelve solo aquellas que sean mayores o iguales a 70.

4. Contar nacionalidades en un array

Crea una función que cuente cuántas veces aparece cada nacionalidad en un array de strings.

5. Comprobar si un nombre está en la clase

Verifica si un nombre específico está presente en un array de compañeros de clase.

6. Convertir temperaturas de Celsius a Fahrenheit

Transforma un array de temperaturas en Celsius a Fahrenheit usando una fórmula matemática.

7. Encontrar el primer número par

Busca y devuelve el primer número par dentro de un array de números.

8. Sumar todos los elementos de un array

Crea una función que sume todos los valores de un array numérico y devuelva el total.

9. Ordenar nombres alfabéticamente

Ordena un array de nombres en orden alfabético ascendente.

10. Multiplicar cada número por su índice

Devuelve un nuevo array donde cada número ha sido multiplicado por su posición (índice) en el array original.
