Lógica de Programación

Aprende a pensar como un programador: paso a paso, con claridad y precisión

¿Qué es la lógica de programación?

La lógica de programación es la base del pensamiento computacional. Permite resolver problemas mediante instrucciones ordenadas que una máquina puede entender.

  • 1. Secuencia: Ejecutar instrucciones en orden, una tras otra.
  • 2. Condicionales: Tomar decisiones según si una condición se cumple (if, else).
  • 3. Repetición: Repetir acciones con bucles (for, while).
  • 4. Variables: Guardar y manipular datos como números, textos o booleanos.
  • 5. Funciones: Agrupar instrucciones que realizan tareas específicas y reutilizables.
  • 6. Entrada y salida: Recibir datos del usuario y mostrar resultados (prompt, alert, console.log).
  • 7. Pensamiento paso a paso: Descomponer problemas en partes pequeñas y manejables.
  • 8. Pseudocódigo: Escribir instrucciones en lenguaje natural antes de codificar.
  • 9. Diagramas de flujo: Representar visualmente el camino lógico de una solución.
  • 10. Prueba y error: Ejecutar, corregir y mejorar el código constantemente.

Ejemplo en pseudocódigo:

Inicio
  Definir edad
  edad ← 18
  Si edad >= 18 Entonces
    Escribir "Eres mayor de edad"
  Sino
    Escribir "Eres menor de edad"
Fin
