¿Qué es la programación?

Explicación extendida para el certificado IFCD0110

Definición

La programación es el proceso de crear instrucciones que una computadora puede entender y ejecutar. Estas instrucciones se escriben en lenguajes de programación y permiten automatizar tareas, resolver problemas y construir aplicaciones.

¿Qué implica programar?

  • Diseñar soluciones lógicas.
  • Escribir código fuente en un lenguaje específico.
  • Probar y depurar errores.
  • Optimizar y documentar el código.

Elementos clave

  • Algoritmo: pasos ordenados para resolver un problema.
  • Código fuente: texto escrito por el programador.
  • Lenguaje: sistema de reglas para escribir instrucciones.
  • Compilador/Intérprete: traduce el código a lenguaje máquina.
  • Depuración: corrección de errores.

Aplicaciones de la programación

  • Desarrollo web y móvil
  • Videojuegos
  • Automatización de procesos
  • Inteligencia artificial
  • Robótica y dispositivos inteligentes

Breve historia

  • 1940s–50s: programación en lenguaje máquina.
  • 1950s–60s: surge el ensamblador.
  • 1970s–90s: nacen C, Pascal, Java.
  • 2000s en adelante: Python, frameworks, IA.

Programar es pensar

La programación desarrolla habilidades como:

  • Pensamiento lógico
  • Resolución de problemas
  • Creatividad
  • Precisión
  • Trabajo en equipo

Ejemplo: función que saluda al usuario


function saludar(nombre) {
  console.log("Hola, " + nombre);
}

saludar("Joaquín");

Este código define una función que muestra un saludo personalizado en la consola.
