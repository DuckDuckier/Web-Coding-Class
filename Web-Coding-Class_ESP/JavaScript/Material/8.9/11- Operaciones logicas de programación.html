🔹 OR (||)

Devuelve 1 si al menos una entrada es 1.


let a = 0;
let b = 1;
console.log(a || b); // 1

🔹 AND (&&)

Devuelve 1 solo si todas las entradas son 1.


let a = 1;
let b = 1;
console.log(a && b); // 1

🔹 NOT (!)

Invierte el valor: si es 0 devuelve 1, si es 1 devuelve 0.


let a = 0;
console.log(!a); // true (equivale a 1)

🔹 XOR (^)

Devuelve 1 si solo una entrada es 1, pero no ambas.


let a = 1;
let b = 0;
console.log(a ^ b); // 1

🔹 NAND (!(a && b))

Devuelve 1 si no todas las entradas son 1.


let a = 1;
let b = 1;
console.log(!(a && b)); // false (equivale a 0)

🔹 NOR (!(a || b))

Devuelve 1 si todas las entradas son 0.


let a = 0;
let b = 0;
console.log(!(a || b)); // true (equivale a 1)

🔹 XNOR (!(a ^ b))

Devuelve 1 si las entradas son iguales.


let a = 1;
let b = 1;
console.log(!(a ^ b)); // true (equivale a 1)
