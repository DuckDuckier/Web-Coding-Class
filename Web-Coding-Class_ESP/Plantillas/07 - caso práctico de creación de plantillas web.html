Examen Práctico: Creación de Plantillas Web

Profesor: Joaquín | Duración: 1 hora

Planteamiento General

Caso práctico: Crea una plantilla web reutilizable a partir de un diseño base. La plantilla debe incluir estructura clara, estilo visual coherente y secciones que puedan adaptarse a distintos proyectos. No es necesario contenido real, pero sí una organización que permita personalización.

1. Plantilla para Portafolio Personal

  • Cabecera con nombre y menú
  • Sección “Sobre mí”
  • Galería de proyectos (mínimo 3 tarjetas)
  • Pie de página con redes sociales

2. Plantilla para Página de Empresa

  • Logo y menú de navegación
  • Sección de servicios (mínimo 3 bloques)
  • Testimonios o clientes
  • Formulario de contacto (estructura sin funcionalidad)

3. Plantilla para Blog Genérico

  • Cabecera con título del blog
  • Lista de entradas (mínimo 2 con imagen y resumen)
  • Barra lateral con categorías
  • Pie de página con aviso legal

4. Plantilla para Landing Page

  • Imagen destacada o banner
  • Texto de presentación
  • Botón de llamada a la acción (CTA)
  • Sección de beneficios o características

5. Plantilla para Curriculum Online

  • Datos personales
  • Experiencia laboral
  • Formación académica
  • Habilidades y contacto
