Estructura de la Plantilla Web

mi-plantilla-web/
├── index.html ← Página principal
├── contacto.html ← Página secundaria
├── servicios.html ← Otra página
├── /css/ ← Estilos
└── estilos.css
├── /js/ ← Scripts
└── funciones.js
├── /img/ ← Imágenes
├── logo.png
└── fondo.jpg
├── /fonts/ ← Tipografías personalizadas
└── roboto.woff2
├── /templates/ ← Fragmentos reutilizables
├── header.html
├── footer.html
└── formulario.html
├── /docs/ ← Documentación del proyecto
└── instrucciones.md
└── README.md ← Descripción general del proyecto

Descripción de Carpetas

Carpeta Contenido Propósito principal
/css/ Archivos de estilos (.css) Define el diseño visual del sitio
/js/ Scripts (.js) Añade interactividad y lógica
/img/ Imágenes (.jpg, .png, .svg) Recursos gráficos para la plantilla
/fonts/ Tipografías (.woff, .ttf) Personalización de texto
/templates/ Fragmentos HTML reutilizables Encabezado, pie, formularios, etc.
/docs/ Documentación del proyecto Guías para el uso o edición de la plantilla
