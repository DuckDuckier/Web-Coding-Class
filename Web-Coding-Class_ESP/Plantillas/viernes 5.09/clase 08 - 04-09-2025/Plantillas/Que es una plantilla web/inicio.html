¿Qué es una plantilla web?

Una plantilla web es una estructura común que se reutiliza en varias páginas del sitio. Aunque el contenido central cambia, el encabezado y el pie de página se mantienen iguales.

En HTML y CSS, como no podemos incluir archivos externos automáticamente, copiamos manualmente el encabezado y el pie en cada página. Esto simula el uso de una plantilla.

El archivo CSS compartido permite que todas las páginas tengan el mismo diseño visual.
