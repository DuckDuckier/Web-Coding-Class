Lenguajes del Desarrollo Web

Ejemplo de descripción

HTML

HTML (HyperText Markup Language) es el lenguaje que estructura el contenido de una página web. Define títulos, párrafos, imágenes, enlaces y más.

Ejemplo: <h1>Hola Mundo</h1>

CSS

CSS (Cascading Style Sheets) se usa para dar estilo a la página web. Controla colores, fuentes, márgenes y diseño visual.

Ejemplo: h1 { color: red; }

JavaScript

JavaScript es un lenguaje de programación que permite interactividad en la web. Se usa para validar formularios, crear efectos dinámicos y más.

Ejemplo: alert("Bienvenido");

PHP

PHP (Hypertext Preprocessor) es un lenguaje que se ejecuta en el servidor. Permite crear páginas dinámicas y conectarse con bases de datos.

Ejemplo: <?php echo "Hola"; ?>

MySQL

MySQL es un sistema de gestión de bases de datos. Se usa para guardar y consultar información como usuarios, productos o mensajes.

Ejemplo: SELECT * FROM usuarios;
