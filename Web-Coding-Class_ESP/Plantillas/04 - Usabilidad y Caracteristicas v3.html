¿Qué es la Usabilidad?

La usabilidad es la capacidad de un sistema para ser fácil de usar, efectivo y agradable para el usuario.

Características principales

Característica Descripción Ejemplo
Eficacia ¿El usuario logra sus objetivos correctamente?
Eficiencia ¿Cuánto tiempo y esfuerzo requiere completar la tarea?
Satisfacción ¿El usuario se siente cómodo y satisfecho al usar el sistema?
Aprendibilidad ¿Es fácil aprender a usar el sistema por primera vez?
Memorabilidad ¿El usuario recuerda cómo usarlo después de un tiempo?
Tolerancia a errores ¿El sistema ayuda a evitar y corregir errores fácilmente?
